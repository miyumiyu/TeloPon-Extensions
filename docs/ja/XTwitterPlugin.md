# 🐦 X (Twitter) 連携プラグイン (XTwitterPlugin.py)

📥 **[XTwitterPlugin.py をダウンロード](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/XTwitterPlugin.py)**

配信のハッシュタグでツイートを定期取得してAIに注入し、AIが配信中にツイートを自動投稿できるプラグインです。  
サムネイル画像やOBSゲーム画面のスクリーンショットを画像付きでツイートすることも可能です。

> **注意:** このプラグインは X (Twitter) の有料APIを使用します。X Developer Portal での開発者登録とAPI従量課金が必要です。

---

## 🌟 主な機能

| 機能 | 説明 |
|---|---|
| ハッシュタグ取得 | 指定ハッシュタグのツイートを定期ポーリングで取得してAIに注入 |
| テキスト投稿 | AIが配信者の指示でツイートを自動投稿（ハッシュタグ・配信URL自動付与） |
| サムネイル付き投稿 | 配信サムネイル画像を添付してツイート |
| ゲーム画面付き投稿 | OBS WebSocket経由でキャプチャした画面を添付してツイート |

## 📋 事前準備: X Developer Portal でAPIキーを取得

### 1. 開発者アカウントの作成

1. [X Developer Portal](https://developer.x.com/) にアクセス
2. Xアカウントでログイン
3. Developer Portal に登録

### 2. アプリの作成と設定

1. 「アプリ」ページでアプリを作成
2. アプリの権限を **「Read and Write」** に設定
3. **重要:** アプリを「Pay Per Use」パッケージに追加（スタンドアロンアプリのまま投稿機能が使えません）

### 3. APIキーの取得

以下の4つのキーを控えてください:

| キー | 場所 |
|---|---|
| **API Key (Consumer Key)** | アプリ設定 → Keys and Tokens |
| **API Secret (Consumer Secret)** | 同上 |
| **Access Token** | 同上 → 「Generate」ボタン |
| **Access Token Secret** | 同上 |

> **重要:** Access Token は「読み取りと書き込み」権限で生成してください。

### 4. クレジットのチャージ

Developer Portal の「請求書作成」→「クレジット」からクレジットカードを登録してチャージしてください。残高がないとAPIリクエストが `402 Payment Required` になります。

## 📋 設定方法

### 1. 設定画面を開く

TeloPonのメイン画面右側、「拡張機能」パネルにある **「🐦 X (Twitter)」** の **「⚙️ 設定」** ボタンをクリックします。

### 2. APIキーを入力

4つのキーをそれぞれ入力し、**「接続テスト」** ボタンをクリック。  
「✅ 認証成功: @あなたのユーザー名」と表示されれば成功です。

### 3. ハッシュタグを設定

- **検索ハッシュタグ**: 取得したいハッシュタグを入力（例: `TeloPon`、`#` は不要）
- **ポーリング間隔**: ツイート取得の間隔（30〜300秒、デフォルト60秒）

### 4. 投稿設定

- **AIによるツイート投稿**: ONにすると配信者の指示でAIがツイート
- **自動付与ハッシュタグ**: 投稿時に自動で付けるハッシュタグ（例: `#TeloPon #配信`）
- **配信URLを自動で付ける**: ONにすると配信プラットフォーム（YouTube/Twitch/ニコニコ）のURLが自動で追加されます

### 5. OBSキャプチャ設定（画像付きツイート用）

ゲーム画面付きツイートを使うには、OBS WebSocket の接続情報を設定します。

| 項目 | 説明 |
|---|---|
| **Host** | OBS WebSocket のアドレス（通常 `127.0.0.1`） |
| **Port** | ポート番号（通常 `4455`） |
| **Password** | OBS WebSocket のパスワード |
| **キャプチャソース名** | スクリーンショットを取るソース名（例: `ゲームキャプチャ`） |

**「📸 テストキャプチャ」** ボタンでプレビュー画像が右側に表示されれば設定成功です。

## ⚠️ ご利用上の注意

- **有料API**: X APIは無料枠が廃止されています。Pay Per Use の従量課金が必要です。
- **Pay Per Use パッケージ**: アプリを「Pay Per Use」に追加しないと投稿は 403 エラーになります。
- **クレジット残高**: チャージがないと 402 エラーになります。
- **ライブ中でも設定変更可能**: TOOL タイプなので配信中でも接続/設定変更できます。
- **ポーリング間隔**: 短くするとコストが増えます。60秒以上を推奨。
- **検索は過去7日間のみ**: X APIの仕様で直近7日のツイートのみ取得可能。
- **ツイート文字数**: 配信URL・ハッシュタグが自動付与されるため、本文は100文字程度を推奨。
- **視聴者コメントからの投稿**: デフォルトでは無効。設定で有効化できます。
