# 🐦 X (Twitter) 連携プラグイン (XTwitterPlugin.py)

📥 **[XTwitterPlugin.py をダウンロード](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/XTwitterPlugin.py)**

配信のハッシュタグでツイートを定期取得してAIに注入し、AIが配信中にツイートを自動投稿できるプラグインです。

> **注意:** このプラグインは X (Twitter) の有料APIを使用します。X Developer Portal での開発者登録とAPI従量課金が必要です。

---

## 🌟 主な機能

| 機能 | 説明 |
|---|---|
| ハッシュタグ取得 | 指定ハッシュタグのツイートを定期ポーリングで取得してAIに注入 |
| ツイート投稿 | AIが配信者の指示でツイートを自動投稿（ハッシュタグ自動付与） |

---

## ⚙️ 必要環境

- X (Twitter) Developer Portal のアプリ（読み取り+書き込み権限）
- 以下のライブラリが必要です:

```
pip install tweepy
```

## 📋 事前準備: X Developer Portal でAPIキーを取得

### 1. 開発者アカウントの作成

1. [X Developer Portal](https://developer.x.com/) にアクセス
2. Xアカウントでログイン
3. Developer Portal に登録（審査が必要な場合あり）

### 2. アプリの作成

1. Dashboard → 「Create Project」または「Create App」
2. アプリの権限を **「Read and Write」** に設定

### 3. APIキーの取得

以下の4つのキーを控えてください:

| キー | 場所 |
|---|---|
| **API Key (Consumer Key)** | アプリ設定 → Keys and Tokens |
| **API Secret (Consumer Secret)** | 同上 |
| **Access Token** | 同上 → 「Generate」ボタン |
| **Access Token Secret** | 同上 |

> **重要:** Access Token は「読み取りと書き込み」権限で生成してください。

## 📋 設定方法

### 1. 設定画面を開く

TeloPonのメイン画面右側、「拡張機能」パネルにある **「X (Twitter) 連携」** の **「⚙️ 設定」** ボタンをクリックします。

### 2. APIキーを入力

4つのキーをそれぞれ入力し、**「接続テスト」** ボタンをクリック。  
「✅ 認証成功: @あなたのユーザー名」と表示されれば成功です。

### 3. ハッシュタグを設定

- **検索ハッシュタグ**: 取得したいハッシュタグを入力（例: `TeloPon`、`#` は不要）
- **ポーリング間隔**: ツイート取得の間隔（30〜300秒、デフォルト60秒）

### 4. 投稿設定

- **AIによるツイート投稿**: ONにすると配信者の指示でAIがツイート
- **自動付与ハッシュタグ**: 投稿時に自動で付けるハッシュタグ（例: `#TeloPon #配信`）

## ⚠️ ご利用上の注意

- **有料API**: X APIは無料枠が廃止されています。従量課金が必要です。
- **ライブ中でも設定変更可能**: TOOL タイプなので配信中でも接続/設定変更できます。
- **ポーリング間隔**: 短くするとコストが増えます。60秒以上を推奨。
- **検索は過去7日間のみ**: X APIの仕様で直近7日のツイートのみ取得可能。
- **視聴者コメントからの投稿**: デフォルトでは無効。設定で有効化できます。
