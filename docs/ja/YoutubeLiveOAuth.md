# 📺 YouTube Live+ (YoutubeLiveOAuth.py)

📥 **[YoutubeLiveOAuth.py をダウンロード](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/YoutubeLiveOAuth.py)**

YouTube Data API v3 + OAuth2 による高機能 YouTube Live 連携プラグインです。  
コメントの読み書き、アンケート、タイトル変更、視聴者数取得などを配信者の音声で操作できます。

> 従来の「YouTube連携ツール (YoutubeLivePlugin.py)」の上位互換です。  
> 従来版はコメント読み取りのみでしたが、本プラグインはOAuth2認証により書き込みや各種操作が可能です。

---

## 🌟 主な機能

| 機能 | 説明 |
|---|---|
| チャットコメント読み取り | 視聴者のコメントをリアルタイムにAIに注入 |
| チャットコメント書き込み | AIがYouTubeのチャットに投稿 |
| アンケート（Poll）作成・締切・集計 | YouTube公式のアンケート機能を音声で操作 |
| 配信タイトル変更 | 音声指示でタイトルを変更 |
| 視聴者数取得 | 現在の同時視聴者数を取得してAIが教えてくれる |
| 配信選択（リスト/URL） | 予約済み配信の一覧から選択、またはURLで直接指定 |
| 権限自動判定 | オーナー/管理者かどうかを自動判定し、使える機能を表示 |

---

## 📋 権限別の機能一覧

接続する配信に対して、あなたがどの権限を持っているかで使える機能が変わります。

| 機能 | オーナー/管理者 | 視聴者/モデレーター |
|---|---|---|
| コメント読み取り | ✅ | ✅ |
| コメント書き込み | ✅ | ✅ |
| 視聴者数取得 | ✅ | ✅ |
| アンケート | ✅ | ❌ |
| タイトル変更 | ✅ | ❌ |

> 権限は接続時に自動判定され、配信情報プレビューに表示されます。  
> 使えない機能は自動的にチェックが外れグレーアウトされます。

## ⚙️ 連携までの流れ

### Step 1: Google Cloud Console の設定（初回のみ）

初回は Google Cloud Console でプロジェクト作成・API有効化・クライアントID取得が必要です。  
画像付きの詳細な手順書を用意していますので、こちらに従って進めてください。

👉 **[Google Cloud Console 設定ガイド（画像付き）](YoutubeLiveOAuth_GCP_Setup.md)**

ガイドの手順:
1. Google Cloud Console にログイン
2. 新しいプロジェクトを作成
3. OAuth同意画面の設定
4. YouTube Data API v3 を有効にする
5. スコープの設定
6. テストユーザーの追加
7. OAuthクライアントIDの作成 → **Client ID** と **Client Secret** を取得
8. TeloPonに入力して認証

ここまで完了すると、TeloPonの設定画面にチャンネル名とサムネイルが表示されます。  
2回目以降は認証済みなので、この手順はスキップできます。

---

### Step 2: 配信を選択して接続する

認証後、配信を選択する方法は2つあります。

#### 方法A: 配信一覧から選択

1. 「配信一覧を取得」ボタンを押す（予約済みの配信が一覧表示されます）
2. リストから配信を選択 → タイトルとサムネイルがプレビューされます
3. 「この配信に接続」ボタンを押す

> ⚠️ **配信一覧には、認証を行ったアカウント自身の配信のみ表示されます。**  
> 他のチャンネルの管理者権限を持っている場合でも、そのチャンネルの配信は一覧に表示されません。  
> 管理者権限で他のチャンネルに接続する場合は、下記の **方法B（URLで直接指定）** をお使いください。

#### 方法B: URLで直接指定

1. YouTube LiveのURLを入力欄に貼り付け
2. 「確認」ボタンを押す → タイトルとサムネイルがプレビューされます
3. 「この配信に接続」ボタンを押す

> どちらの方法でも、接続時に権限が自動判定されます。  
> `権限: オーナー（全機能利用可能）` または `権限: 視聴者（コメント・視聴者数のみ）` と表示されます。

---

### Step 3: ライブ配信を開始する

TeloPonのメイン画面で **「🔴 ライブ接続開始」** を押します。

> **重要:** ライブ配信を開始する**前に**「この配信に接続」を押しておいてください。  
> 先に接続しておくと、AIに配信情報やコマンドの説明が確実に伝わります。  
> ライブ開始後に接続することも可能ですが、コマンドの認識精度が若干低下する可能性があります。

## 🎙️ 発話例（こう言うとAIが反応します）

### コメント書き込み

配信者がAIにYouTubeチャットへの書き込みを指示できます。

| 配信者の発話 | AIの動作 |
|---|---|
| 「チャットに"こんにちは"って書いて」 | YouTubeチャットに「こんにちは」と投稿します |
| 「"今日もよろしく"ってコメントして」 | YouTubeチャットに「今日もよろしく」と投稿します |

### アンケート ※オーナー/管理者のみ

YouTubeの公式アンケート機能（Poll）を使って、視聴者に投票してもらえます。  
選択肢は2〜4個まで設定できます。

| 配信者の発話 | AIの動作 |
|---|---|
| 「好きな色をアンケートして」 | 「好きな色は？ 赤 / 青 / 緑」のようなアンケートがチャットに表示されます |
| 「次に何やるかアンケート取って」 | AIが適切な選択肢を考えてアンケートを作成します |
| 「集計して」「アンケート締め切って」 | アンケートを締め切り、AIが結果を視聴者に発表します |

> アンケートを締め切ると、AIに集計結果がフィードバックされます。  
> AIが「赤が15票で1位です！」のようにテロップで結果を発表します。

### タイトル変更 ※オーナー/管理者のみ

| 配信者の発話 | AIの動作 |
|---|---|
| 「タイトルを"雑談配信"に変えて」 | 配信タイトルが「雑談配信」に変更されます |
| 「タイトル変えて、マイクラ建築回にして」 | 配信タイトルが「マイクラ建築回」に変更されます |

### 視聴者数取得

| 配信者の発話 | AIの動作 |
|---|---|
| 「今何人見てる？」 | AIが現在の同時視聴者数を取得し、テロップで教えてくれます |
| 「視聴者数教えて」 | 同上 |

## 🛡️ コマンドの視聴者許可設定

各機能には「視聴者可」チェックボックスがあります。  
**視聴者のコメントからもAIにコマンドを実行させるかどうか**を制御できます。

| 設定 | 動作 |
|---|---|
| ☑ 視聴者可 ON | 視聴者が「アンケートして」等とコメントしたらAIが実行することがあります |
| ☐ 視聴者可 OFF | 配信者の音声指示のみで実行。視聴者コメントからは実行しません |

> タイトル変更など影響が大きい操作を視聴者に許可するかは慎重に判断してください。

## 🔧 機能設定

設定画面の右側に機能のON/OFFチェックボックスがあります。

| 機能 | デフォルト | 説明 |
|---|---|---|
| コメント取得 | ☑ ON | 視聴者のコメントをAIに送る |
| コメント書き込み | ☑ ON | AIがチャットに投稿できる |
| アンケート | ☑ ON | アンケートの作成・締切 |
| タイトル変更 | ☑ ON | 配信タイトルの変更 |

- ライブ配信中は機能設定がグレーアウトされ変更できません（注入済みのプロンプトは取り消せないため）
- 権限がない機能は自動的にチェックが外れグレーアウトされます

設定画面は左右2列のレイアウトになっています。

### 左列

| セクション | 内容 |
|---|---|

## ⚠️ ご利用上の注意点

### 接続のタイミング

ライブ配信を開始する**前に**「この配信に接続」を押しておくことを推奨します。  
ライブ開始後に接続した場合でもコマンドは使えますが、AIへの伝わり方が若干異なります。

### OAuth同意画面のテストモード

Google Cloud Consoleで作成したOAuthアプリは「テストモード」のままで使えます。  
テストユーザーに自分のメールアドレスを追加すれば、公開審査は不要です。  
テストモードでは最大100人のテストユーザーを登録できます。

### 管理者権限での接続

配信チャンネルのオーナーでなくても、YouTube Studioで「管理者」として招待されていれば全機能が使えます。  
ただし、認証時に管理対象のBrand Accountを選択してログインする必要があります。

### APIクォータ

YouTube Data API v3には1日10,000ユニットのクォータ制限があります。  
通常の配信では十分な量ですが、大量のAPI呼び出しが発生する場合は注意してください。

### 従来のYouTube連携ツールとの併用

従来版（YoutubeLivePlugin.py）と本プラグイン（YoutubeLiveOAuth.py）を同時に有効にすると、  
コメントが二重にAIに送信されます。どちらか一方のみを有効にしてください。
