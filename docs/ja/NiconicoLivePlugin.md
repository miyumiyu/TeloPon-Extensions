# 📺 ニコニコ生放送連携プラグイン (NiconicoLivePlugin.py)

📥 **[NiconicoLivePlugin.py をダウンロード](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/NiconicoLivePlugin.py)**

ニコニコ生放送のコメント・ギフト・ニコニ広告・アンケート・統計情報をリアルタイムに取得し、AIに注入するプラグインです。  
ログインなしでもコメント読み取りが可能で、ログインすればアンケート作成や運営コメント投稿などの操作機能も使えます。

---

## 🌟 主な機能

### 📖 基本機能（ログイン不要）

| 機能 | 説明 |
|---|---|
| コメント読み取り | 視聴者のコメントをリアルタイムに取得してAIに注入 |
| ギフト通知 | ギフトが届くとAIが感謝の言葉を返す |
| ニコニ広告通知 | 広告されるとAIがお礼を述べる |
| エモーション通知 | エモーションが大量に送られると盛り上がりに反応 |
| 統計情報更新 | 来場者数がマイルストーンを超えると自動通知 |
| アンケート結果通知 | アンケートの結果をAIがコメント |

### ✏️ 操作機能（ログイン必要・自分の配信のみ）

| 機能 | 説明 |
|---|---|
| アンケート作成/結果/終了 | AIの音声指示でアンケートを作成・集計・終了 |
| 運営コメント投稿/削除 | 運営コメントの投稿（色・固定対応）と削除 |
| 視聴者数取得 | 現在の同時視聴者数とコメント数を取得 |

## 📋 設定方法

### 1. 設定画面を開く

TeloPonのメイン画面右側、「拡張機能」パネルにある **「ニコニコ生放送連携」** の **「⚙️ 設定」** ボタンをクリックします。

### 3. ニコニコ生放送に接続

1. ニコニコ生放送のURLを入力します（例: `https://live.nicovideo.jp/watch/lv350259676`）
2. **「接続」** ボタンをクリック
3. 配信情報（タイトル・説明文・サムネイル）が表示されれば成功です

> まだ始まっていない配信の場合は「配信開始待機中」と表示され、配信が始まると自動で接続されます。

### 4. ログイン（任意）

アンケート作成や運営コメント投稿などの操作機能を使うには、ニコニコアカウントでログインが必要です。

1. 設定画面上部の「ニコニコ ログイン」セクションにメールアドレスとパスワードを入力
2. **「ログイン」** ボタンをクリック
3. 二段階認証が有効な場合は、確認コード（6桁）の入力画面が表示されます
4. 認証が完了するとプロフィールアイコンと名前が表示されます

> **操作機能は自分の配信でのみ使用可能です。** 他人の配信では読み取り専用になります。

### 5. 機能のON/OFF設定

右カラムで各機能を個別にON/OFFできます。

**読み取り機能（ログイン不要）:**
- コメント読み取り
- ギフト通知
- ニコニ広告通知
- エモーション通知
- 統計情報更新
- アンケート結果通知

**操作機能（要ログイン）:**
- 視聴者数取得（「視聴者可」チェックで視聴者コメントからも実行可能）
- アンケート作成/終了
- 運営コメント投稿

## 🎤 音声コマンド一覧

配信中に以下のように話しかけると、AIがコマンドを実行します。

### アンケート

| 音声例 | 動作 |
|---|---|
| 「アンケートして」「投票して」 | AIが設問と選択肢を考えてアンケート作成 |
| 「集計して」「結果教えて」 | アンケート結果を表示 |
| 「アンケート締め切って」 | アンケートを終了 |

### 運営コメント

| 音声例 | 動作 |
|---|---|
| 「運営コメントで○○って出して」 | 運営コメントを投稿 |
| 「赤で○○って出して」 | 色付き運営コメント（white/red/pink/orange/yellow/green/cyan/blue/purple/black） |
| 「固定で○○って出して」 | 固定（常時表示）運営コメント |
| 「運営コメント消して」 | 運営コメントを削除 |

### 情報取得

| 音声例 | 動作 |
|---|---|
| 「今何人？」「視聴者数は？」 | 現在の同時視聴者数とコメント数を取得 |

## 📝 AI注入の仕組み

### コメント
視聴者のコメントは一定間隔でバッチ処理され、まとめてAIに送信されます。匿名ユーザーの表示名は設定で変更可能です。

### ギフト・ニコニ広告
ギフトやニコニ広告が検知されると、送り主の名前・アイテム名・ポイント数がAIに通知され、感謝のコメントを促します。

### 統計マイルストーン
来場者数が 100 / 500 / 1,000 / 2,000 / 3,000 / 5,000 / 10,000 / 20,000 / 50,000 / 100,000 を超えると、AIに通知されます。

### 配信情報
接続時に配信のタイトルと説明文がプロンプトに追加され、AIが番組の内容を把握した上でトークを進めます。

---

## ⚠️ ご利用上の注意

- **ライブ中でも接続・切断可能:** このプラグインは TOOL タイプなので、配信中でも接続/切断をリアルタイムに行えます。
- **操作機能は自分の配信のみ:** ログインしていても、他人の配信ではアンケート作成や運営コメント投稿はできません（自動でグレーアウトされます）。
- **二段階認証対応:** ニコニコアカウントの二段階認証（2FA）に対応しています。
