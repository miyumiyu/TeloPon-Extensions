# 🔊 テロップ読み上げ - Gemini TTS (GeminiTTS.py)

📥 **[GeminiTTS.py をダウンロード](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/GeminiTTS.py)**

最新の **Gemini 3.1 Flash TTS Preview** を使って、AIが出力したテロップを高品質な音声で読み上げるプラグインです。  
TeloPon本体に設定済みのGemini APIキーをそのまま使用するため、**追加のアカウント登録や認証は不要**です。

---

## 🌟 主な機能

| 機能 | 説明 |
|---|---|
| テロップ自動読み上げ | AIが出力したMAINテキストを自動で音声化 |
| 30種類のプリセットボイス | Kore / Puck / Zephyr / Charon / Fenrir など多彩なキャラクター |
| 感情（badge）連動 | AIが付与した感情タグに応じて読み方が変わる |
| キャラ指示（自由入力） | 「ツンデレ風」「元気な女の子」「落ち着いた紳士」などの演出指示 |
| 読み上げスピード調整 | ゆっくり / 普通 / 早口 |
| 再生デバイス選択 | 出力先デバイスを指定可能（仮想デバイス対応） |
| 音量調整 | スライダーで0〜100% |
| ディレイ設定 | テロップ表示から読み上げ開始までの待ち時間を調整 |
| TOPIC読み上げ | 見出し（TOPIC）も読み上げるかどうかを切替 |
| モデル選択 | Gemini 3.1 Flash TTS Preview / Gemini 2.5 Flash Preview TTS |

---

## ⚠️ 重要な注意事項

> **Gemini TTS APIはレート制限が厳しく、無料APIの場合は5回ほどでリミットに到達します。**
>
> 長時間配信での常用には適さないため、**お試し・デモ用途**としてご利用ください。  
> 本格利用の場合は、Gemini APIの有料プラン（従量課金）をご検討ください。

レート制限に到達すると、プラグインのステータスに「⚠️ リミットに到達（○秒後に再試行してください）」と表示され、自動的にキューをクリアして指定時間待機します。

## ⚙️ 設定方法

### 1. 前提条件
TeloPon本体で **Gemini APIキー** が設定済みであること。  
このプラグインは本体の設定をそのまま流用します。

### 2. 設定画面を開く
TeloPonのメイン画面右側、「拡張機能」パネルにある **「Gemini TTS」** の **「⚙️ 設定」** ボタンをクリックします。

### 3. 各項目を設定する

| 項目 | 説明 |
|---|---|
| **ON / OFF** | チェックを入れると読み上げが有効になります |
| **TOPICも読み上げる** | テロップの見出し（TOPIC）も読み上げる場合はON |
| **音声** | 30種類のプリセットボイスから選択 |
| **再生デバイス** | 音声の出力先（VB-CABLE等の仮想デバイス選択可） |
| **キャラ指示** | 自由入力。例: `ツンデレ風`、`元気な女の子`、`落ち着いた紳士` 等 |
| **スピード** | ゆっくり / 普通 / 早口 |
| **音量** | 0〜100% |
| **ディレイ** | テロップ表示から読み上げ開始までの秒数（0〜10秒） |

### 4. テスト再生
「▶ テスト再生」ボタンで設定内容のテスト音声を確認できます。

## 🎭 表現力について

Gemini TTSは自然言語での演出指示に対応しています。  
キャラ指示に入力した内容がそのままプロンプトに反映されます。

### キャラ指示の例

```
ツンデレ風
→ 「ツンデレ風、喜びな感じで、テンポよく早口で読んでください: テキスト」
```

AIが付与した `[BDG]` 感情タグも自動で組み合わされます:

- `[BDG]喜び` + キャラ「ツンデレ風」 + スピード「早口」
- → 「ツンデレ風、喜びな感じで、テンポよく早口で」のプロンプト

キャラ指示は空欄でも動作します（その場合は感情・スピードのみ反映）。

## 🎧 OBSに音声を取り込む方法

配信にAIの読み上げ音声を乗せたい場合、仮想オーディオデバイスを使います。

1. **VB-CABLE** などの仮想オーディオデバイスをインストール
2. プラグインの「再生デバイス」で仮想デバイス（例: `CABLE Input`）を選択
3. OBSで「音声入力キャプチャ」を追加し、仮想デバイス（例: `CABLE Output`）を選択

---

## 💡 声（ボイス）の選び方

30種類のプリセットから選択可能です。日本語でも英語でも自動的に適切な発音で読み上げます。

代表的なボイス:
- **Kore** — 落ち着いた女性声（デフォルト）
- **Puck** — 明るく活発な声
- **Zephyr** — 柔らかく穏やかな声
- **Charon** — 深みのある男性声
- **Fenrir** — 力強い男性声

好みのボイスが見つからない場合は、テスト再生ボタンで順に聞き比べてください。

## ⚠️ ご利用上の注意

- **ライブ中でも設定変更可能:** TOOL タイプなので、配信中に音声・キャラ指示・スピード等をリアルタイムに変更できます。
- **レート制限に注意:** 無料プランでは短時間に何度も発動すると制限がかかります。上記の注意事項を参照してください。
- **ストリーミング非対応:** Gemini TTS APIはストリーミング再生に対応していないため、テロップ出力から音声再生まで1〜3秒の遅延があります。
