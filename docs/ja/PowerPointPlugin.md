# 📊 PowerPoint操作プラグイン (PowerPointPlugin.py)

📥 **[PowerPointPlugin.py をダウンロード](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/PowerPointPlugin.py)**

配信者の音声指示でPowerPointのスライドショーを操作できるプラグインです。  
スライドの送り/戻し・特定ページへのジャンプ・ブラックアウトなどに対応し、スライドが変わるとタイトルやノートをAIに自動注入します。

---

## 🌟 主な機能

| 機能 | 説明 |
|---|---|
| スライド送り/戻し | 「次のスライド」「前に戻って」で操作 |
| ページジャンプ | 「3ページ目に飛んで」で特定スライドへ移動 |
| 最初/最後へ移動 | 「最初に戻って」「最後のスライド」で先頭/末尾へ |
| プレゼン開始/終了 | 「プレゼン始めて」「プレゼン終わり」でスライドショーを制御 |
| 画面ブラックアウト | 「画面消して」で暗転、「画面戻して」で復帰 |
| スライドノート注入 | スライドが変わると、タイトルとノートをAIに自動送信（カンペとして利用可能） |

---

## ⚙️ 必要環境

- Windows PC
- Microsoft PowerPoint がインストールされていること
- pywin32（TeloPon exe版には同梱済み）

---

## 📋 設定方法

### 1. 設定画面を開く

TeloPonのメイン画面右側、「拡張機能」パネルにある **「PowerPoint操作」** の **「⚙️ 設定」** ボタンをクリックします。

### 2. PowerPointに接続する

1. 操作したい PowerPoint ファイルを事前に開いておきます
2. 設定画面の **「PowerPointに接続」** ボタンをクリック
3. 「接続中: ファイル名.pptx」と表示されれば成功です

### 3. オプションを設定する

| 項目 | 説明 |
|---|---|
| **PowerPoint連携 ON** | チェックを入れるとプラグインが有効になります |
| **スライドのノートをAIに注入** | ONにすると、スライドが変わった際にノート（発表者ノート）をAIに送信します |
| **スライドのタイトルをAIに注入** | ONにすると、スライドのタイトルテキストをAIに送信します |

### 4. テストボタン

「次へ（テスト）」「前へ（テスト）」ボタンで、スライドショー中に動作を確認できます。

## 🎤 音声コマンド一覧

配信中に以下のように話しかけると、AIがスライドを操作します。

| 音声例 | 動作 |
|---|---|
| 「次のスライド」「次へ」「次のページ」 | 次のスライドへ |
| 「前のスライド」「戻って」「前に」 | 前のスライドへ |
| 「3ページ目に飛んで」「5番に行って」 | 指定ページへジャンプ |
| 「最初に戻って」 | 最初のスライドへ |
| 「最後のスライド」 | 最後のスライドへ |
| 「プレゼン始めて」 | スライドショー開始 |
| 「プレゼン終わり」 | スライドショー終了 |
| 「画面消して」「ブラックアウト」 | 画面を暗転 |
| 「画面戻して」 | 暗転から復帰 |

## 📝 スライドノートのAI注入について

スライドが切り替わるたびに、以下の情報がAIに自動送信されます。

```
【プレゼン資料のカンペ】
スライド 3/15: スライドタイトル
ここにスライドのノートテキストが入ります...
```

これにより、AIが現在のスライドの内容を把握し、プレゼンの流れに沿った相槌やコメントを返すことができます。  
発表者ノートに「ここで笑いを取る」「このポイントを強調」などのメモを書いておくと、AIがそれを参考にしてリアクションします。

## ⚠️ ご利用上の注意

- **ライブ中でも接続・切断可能:** このプラグインは TOOL タイプなので、配信中でもPowerPointへの接続/切断をリアルタイムに行えます。
- **Windowsのみ対応:** COM経由でPowerPointを操作するため、Windows専用です。
- **PowerPointを先に起動:** 接続前にPowerPointでファイルを開いておく必要があります。
- **スライドショーモード:** スライドの送り/戻しなどの操作は、スライドショーモード中のみ有効です。
- **視聴者コメントからの操作:** セキュリティ上、視聴者コメントからのスライド操作は無効です（配信者の音声のみ）。
