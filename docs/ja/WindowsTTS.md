# 🔊 Windows 読み上げプラグイン (WindowsTTS.py)

Windows標準の音声合成エンジン（SAPI5）を使って、AIが出力したテロップのテキストを自動的に読み上げるプラグインです。  
追加の外部ソフトは不要で、Windowsに標準搭載されている音声がそのまま使えます。

> 必要ライブラリ: `pip install pyttsx3`

---

## 🌟 主な機能

| 機能 | 説明 |
|---|---|
| テロップ自動読み上げ | AIが出力したMAINテキストを自動で音声化 |
| 音声選択 | Windows にインストール済みの音声から選択（日本語/英語等） |
| 再生デバイス選択 | 出力先デバイスを指定可能（VB-CABLE等の仮想デバイスにも対応） |
| 速度調整 | 読み上げ速度をスライダーで調整 |
| ピッチ調整 | 声の高さをスライダーで調整 |
| 音量調整 | 読み上げ音量をスライダーで調整 |
| TOPIC読み上げ | テロップの見出し（TOPIC）も読み上げるかどうかを切替 |
| 絵文字除去 | Unicode絵文字や `:emoji:` 形式のテキストを自動的に読み上げから除外 |

---

## ⚙️ 設定方法

### 1. 設定画面を開く

TeloPonのメイン画面右側、「拡張機能」パネルにある **「Windows 読み上げ」** の **「⚙️ 設定」** ボタンをクリックします。

### 2. 各項目を設定する

<img src="../images/WindowsTTS.png" width="400">

| 項目 | 説明 |
|---|---|
| **読み上げ ON** | チェックを入れると読み上げが有効になります |
| **音声** | 読み上げに使う音声を選択します。日本語の場合は「Microsoft Haruka Desktop - Japanese」がよく使われます |
| **再生デバイス** | 音声の出力先デバイスを選びます。「既定のデバイス」はWindowsで設定されたデフォルト出力です |
| **速度** | 読み上げの速さを調整します（100=ゆっくり〜350=速い、デフォルト200） |
| **ピッチ** | 声の高さを調整します（-10=低い〜+10=高い、デフォルト0） |
| **音量** | 読み上げの音量を調整します（0〜100、デフォルト100） |
| **TOPICも読み上げる** | チェックを入れると、テロップの見出し（TOPIC）も読み上げます |

### 3. テスト読み上げ

「テスト読み上げ」ボタンを押すと、設定した内容でテスト音声が再生されます。  
問題なければ「閉じる」で設定を保存します。

## 🎧 OBSに音声を取り込む方法

OBSの配信にAIの読み上げ音声を乗せたい場合、仮想オーディオデバイスを使います。

### 手順

1. **VB-CABLE** などの仮想オーディオデバイスをインストール
2. WindowsTTSプラグインの「再生デバイス」で仮想デバイス（例: `CABLE Input`）を選択
3. OBSで「音声入力キャプチャ」を追加し、仮想デバイス（例: `CABLE Output`）を選択

これでAIの読み上げ音声がOBSに取り込まれ、配信に乗ります。

---

## 📝 絵文字の除去について

テロップに含まれる絵文字は読み上げから自動的に除外されます。

| 種類 | 例 | 動作 |
|---|---|---|
| Unicode絵文字 | 🎮 🔥 💬 | 除去（読み上げない） |
| テキスト絵文字 | `:fire:` `:thumbsup:` | 除去（読み上げない） |
| 通常のテキスト | こんにちは | そのまま読み上げ |

---

## ⚠️ ご利用上の注意

- **ライブ中でも設定変更可能:** このプラグインは TOOL タイプなので、配信中でも速度・音声・デバイスなどをリアルタイムに変更できます。
- **Windowsのみ対応:** SAPI5はWindows専用の音声合成エンジンです。macOS/Linuxでは動作しません。
- **音声の追加:** Windows の「設定」→「時刻と言語」→「音声認識」から追加の音声パックをインストールできます。
