# 📺 YouTube Live+ (YoutubeLiveOAuth.py)

📥 **[Скачать YoutubeLiveOAuth.py](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/YoutubeLiveOAuth.py)**

Многофункциональный плагин интеграции с YouTube Live на основе YouTube Data API v3 + OAuth2.  
Позволяет управлять чтением и записью комментариев, опросами, изменением заголовка и получением количества зрителей голосом стримера.

> Является улучшенной версией предыдущего плагина «YouTube Live (YoutubeLivePlugin.py)».  
> Предыдущая версия поддерживала только чтение комментариев, а данный плагин благодаря авторизации OAuth2 позволяет также отправлять сообщения и выполнять различные операции.

---

## 🌟 Основные функции

| Функция | Описание |
|---|---|
| Чтение комментариев чата | Комментарии зрителей передаются ИИ в реальном времени |
| Отправка комментариев в чат | ИИ публикует сообщения в чат YouTube |
| Создание, закрытие и подсчёт опросов (Poll) | Управление официальной функцией опросов YouTube голосом |
| Изменение заголовка трансляции | Изменение заголовка по голосовой команде |
| Получение количества зрителей | ИИ сообщает текущее количество одновременных зрителей |
| Выбор трансляции (список/URL) | Выбор из списка запланированных трансляций или указание URL напрямую |
| Автоматическое определение прав | Автоматически определяет, являетесь ли вы владельцем/администратором, и отображает доступные функции |

## ⚙️ Порядок настройки интеграции

### Шаг 1: Настройка Google Cloud Console (только при первом использовании)

При первом использовании необходимо создать проект, включить API и получить Client ID в Google Cloud Console.  
Мы подготовили подробное руководство с изображениями — следуйте ему.

👉 **[Руководство по настройке Google Cloud Console (с изображениями)](YoutubeLiveOAuth_GCP_Setup.md)**

Этапы руководства:
1. Войдите в Google Cloud Console
2. Создайте новый проект
3. Настройте экран согласия OAuth
4. Включите YouTube Data API v3
5. Настройте области доступа (Scopes)
6. Добавьте тестовых пользователей
7. Создайте OAuth Client ID → получите **Client ID** и **Client Secret**
8. Введите данные в TeloPon и пройдите авторизацию

После выполнения этих шагов на экране настроек TeloPon отобразятся название канала и миниатюра.  
При последующих использованиях авторизация уже пройдена, поэтому этот шаг можно пропустить.

---

### Шаг 2: Выбор и подключение к трансляции

После авторизации есть два способа выбрать трансляцию.

#### Способ A: Выбор из списка трансляций

1. Нажмите кнопку «Получить список трансляций» (отобразятся запланированные трансляции)
2. Выберите трансляцию из списка → заголовок и миниатюра отобразятся в предпросмотре
3. Нажмите кнопку «Подключиться к этой трансляции»

> ⚠️ **В списке трансляций отображаются только трансляции аккаунта, с которого была выполнена авторизация.**  
> Даже если у вас есть права администратора на другом канале, его трансляции не будут показаны в списке.  
> Для подключения к другому каналу с правами администратора используйте **Способ B (указание URL напрямую)**.

#### Способ B: Указание URL напрямую

1. Вставьте URL трансляции YouTube Live в поле ввода
2. Нажмите кнопку «Проверить» → заголовок и миниатюра отобразятся в предпросмотре
3. Нажмите кнопку «Подключиться к этой трансляции»

> При любом способе права определяются автоматически при подключении.  
> Отображается `Права: Владелец (все функции доступны)` или `Права: Зритель (только комментарии и количество зрителей)`.

---

### Шаг 3: Запуск прямой трансляции

Нажмите **«🔴 Начать прямую трансляцию»** на главном экране TeloPon.

> **Важно:** Рекомендуется нажать «Подключиться к этой трансляции» **до** начала прямой трансляции.  
> При предварительном подключении информация о трансляции и описание команд будут точно переданы ИИ.  
> Подключение после начала трансляции также возможно, но точность распознавания команд может немного снизиться.

## 🎙️ Примеры голосовых команд (ИИ реагирует на подобные фразы)

### Отправка комментариев

Стример может дать ИИ команду написать в чат YouTube.

| Фраза стримера | Действие ИИ |
|---|---|
| «Напиши в чат "Привет"» | ИИ отправит «Привет» в чат YouTube |
| «Напиши комментарий "Спасибо за просмотр"» | ИИ отправит «Спасибо за просмотр» в чат YouTube |

### Опросы ※Только для владельца/администратора

Используя официальную функцию опросов YouTube (Poll), можно предложить зрителям проголосовать.  
Можно задать от 2 до 4 вариантов ответа.

| Фраза стримера | Действие ИИ |
|---|---|
| «Создай опрос о любимом цвете» | В чате появится опрос вида «Какой ваш любимый цвет? Красный / Синий / Зелёный» |
| «Проведи опрос, чем заняться дальше» | ИИ подберёт подходящие варианты и создаст опрос |
| «Подведи итоги» / «Закрой опрос» | ИИ закрывает опрос и объявляет результаты зрителям |

> При закрытии опроса результаты передаются ИИ.  
> ИИ объявит результаты на баннере, например: «Красный на первом месте с 15 голосами!»

### Изменение заголовка ※Только для владельца/администратора

| Фраза стримера | Действие ИИ |
|---|---|
| «Измени заголовок на "Разговорный стрим"» | Заголовок трансляции изменится на «Разговорный стрим» |
| «Смени заголовок, пусть будет "Строим в Minecraft"» | Заголовок трансляции изменится на «Строим в Minecraft» |

### Получение количества зрителей

| Фраза стримера | Действие ИИ |
|---|---|
| «Сколько сейчас зрителей?» | ИИ получит текущее количество одновременных зрителей и покажет на баннере |
| «Покажи количество зрителей» | То же самое |

## 🛡️ Настройка разрешений команд для зрителей

Для каждой функции есть флажок «Доступно зрителям».  
**Управляет тем, может ли ИИ выполнять команды по комментариям зрителей.**

| Настройка | Поведение |
|---|---|
| ☑ Доступно зрителям ВКЛ | ИИ может выполнить команду, если зритель напишет «создай опрос» и т.п. в комментариях |
| ☐ Доступно зрителям ВЫКЛ | Выполняется только по голосовой команде стримера. Комментарии зрителей не запускают команды |

> Будьте осторожны с разрешением зрителям использовать функции с большим влиянием, такие как изменение заголовка.

## ⚠️ Примечания по использованию

### Время подключения

Рекомендуется нажать «Подключиться к этой трансляции» **до** начала прямой трансляции.  
При подключении после начала трансляции команды всё равно будут работать, но способ их передачи ИИ будет немного отличаться.

### Тестовый режим экрана согласия OAuth

Приложение OAuth, созданное в Google Cloud Console, может работать в «тестовом режиме».  
Достаточно добавить свой email в тестовые пользователи — публичная проверка не требуется.  
В тестовом режиме можно зарегистрировать до 100 тестовых пользователей.

### Подключение с правами администратора

Даже если вы не являетесь владельцем канала, все функции доступны, если вы приглашены как «администратор» в YouTube Studio.  
Однако при авторизации необходимо выбрать управляемый Brand Account и войти с его помощью.

### Квота API

YouTube Data API v3 имеет ограничение в 10 000 единиц квоты в день.  
Для обычных трансляций этого достаточно, но будьте внимательны при большом количестве вызовов API.

### Совместное использование с предыдущим плагином YouTube

При одновременном включении предыдущей версии (YoutubeLivePlugin.py) и данного плагина (YoutubeLiveOAuth.py)  
комментарии будут отправляться ИИ в двойном экземпляре. Включайте только один из плагинов.
