# 📊 Плагин управления PowerPoint (PowerPointPlugin.py)

📥 **[Скачать PowerPointPlugin.py](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/PowerPointPlugin.py)**

Плагин для управления слайд-шоу PowerPoint голосовыми командами во время трансляции.  
Поддерживает переключение слайдов, переход к определённой странице, затемнение экрана и автоматическую передачу заголовков и заметок слайдов ИИ.

---

## 🌟 Основные функции

| Функция | Описание |
|---|---|
| Следующий/предыдущий слайд | "Следующий слайд", "Назад" для навигации |
| Переход к странице | "Перейди на страницу 3" для перехода к определённому слайду |
| Первый/последний слайд | "Вернись к началу", "Последний слайд" |
| Начать/завершить презентацию | "Начни презентацию", "Заверши презентацию" |
| Затемнение экрана | "Погаси экран" для затемнения, "Верни экран" для восстановления |
| Передача заметок слайда | Автоматическая отправка заголовка и заметок ИИ при смене слайда |

---

## ⚙️ Требования

- ПК с Windows
- Установленный Microsoft PowerPoint
- pywin32 (включён в exe-версию TeloPon)

---

## 📋 Настройка

### 1. Откройте настройки

Нажмите кнопку **"⚙️ Настройки"** рядом с **"Управление PowerPoint"** на панели расширений справа на главном экране TeloPon.

### 2. Подключитесь к PowerPoint

1. Заранее откройте файл PowerPoint, которым хотите управлять
2. Нажмите кнопку **"Подключить к PowerPoint"** в окне настроек
3. Если отображается "Подключено: имя_файла.pptx" — подключение успешно

### 3. Настройте параметры

| Параметр | Описание |
|---|---|
| **PowerPoint ВКЛ** | Установите флажок для активации плагина |
| **Передавать заметки слайда ИИ** | При включении отправляет заметки докладчика ИИ при смене слайда |
| **Передавать заголовок слайда ИИ** | При включении отправляет заголовок слайда ИИ при смене слайда |

### 4. Тестовые кнопки

Используйте кнопки "Далее (Тест)" и "Назад (Тест)" для проверки работы во время слайд-шоу.

---

## 🎤 Голосовые команды

Во время трансляции говорите следующее для управления слайдами:

| Голосовая команда | Действие |
|---|---|
| "Следующий слайд" / "Дальше" | Перейти к следующему слайду |
| "Предыдущий слайд" / "Назад" | Перейти к предыдущему слайду |
| "Перейди на страницу 3" | Перейти к указанной странице |
| "Вернись к началу" | Перейти к первому слайду |
| "Последний слайд" | Перейти к последнему слайду |
| "Начни презентацию" | Начать слайд-шоу |
| "Заверши презентацию" | Завершить слайд-шоу |
| "Погаси экран" / "Затемнение" | Затемнить экран |
| "Верни экран" | Восстановить экран |

## 📝 Передача заметок слайда ИИ

При каждой смене слайда ИИ автоматически получает следующую информацию:

```
[Заметки презентации]
Слайд 3/15: Заголовок слайда
Текст заметок докладчика...
```

Это позволяет ИИ понимать содержание текущего слайда и давать соответствующие комментарии по ходу презентации.  
Добавьте в заметки указания вроде "подчеркни этот момент" или "пошути здесь", и ИИ будет учитывать их в своих реакциях.

---

## ⚠️ Примечания

- **Подключение/отключение во время трансляции:** Плагин типа TOOL — подключение и отключение PowerPoint возможно в реальном времени.
- **Только Windows:** Управление PowerPoint через COM работает только в Windows.
- **Сначала откройте PowerPoint:** Перед подключением необходимо открыть файл PowerPoint.
- **Режим слайд-шоу:** Команды навигации работают только в активном слайд-шоу.
- **Команды зрителей отключены:** В целях безопасности зрители не могут управлять слайдами через комментарии (только голос стримера).
