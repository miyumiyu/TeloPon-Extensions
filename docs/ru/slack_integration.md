# 🏢 Интеграция с комментариями Slack (slack_integration.py)

С помощью этого плагина ИИ TeloPon сможет без задержки озвучивать комментарии, опубликованные в рабочем пространстве Slack вашей компании или каналах команды!
Благодаря использованию современной технологии «Socket Mode» реакция происходит удивительно быстро в режиме реального времени.

Настройка состоит из трёх основных шагов: **«① Установка плагина»**, **«② Создание бота на стороне Slack (получение двух ключей)»** и **«③ Настройка в интерфейсе TeloPon»**. Шагов немного больше, но при последовательном выполнении настройка пройдёт без проблем!

---

## 📥 Шаг 1: Установка плагина и подготовка

📥 **[Скачать slack_integration.py](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/slack_integration.py)**

1. Скачайте файл по ссылке выше.
2. Поместите скачанный файл в папку **`plugins`** внутри основной папки TeloPon.
3. Запустите TeloPon. Если в списке «🔌 Расширения» на главном экране появился пункт **«🏢 Интеграция с комментариями Slack»**, установка прошла успешно!

---

## 🛠️ Шаг 2: Создание бота Slack и получение «двух токенов»

Чтобы ИИ мог считывать комментарии Slack, необходимо создать специального «бота (приложение)». На этом этапе вы получите **2 вида токенов (ключей)**.

### 1. Создание приложения и присвоение имени
1. Перейдите в браузере на ПК на [Slack API (Your Apps)](https://api.slack.com/apps) и нажмите **«Create New App»** в правом верхнем углу.
2. Выберите **«From scratch»**, введите любое имя (например: `TeloPon Bot`) и выберите рабочее пространство для установки, затем нажмите **«Create App»**.
3. **[Важно]** Откройте **«App Home»** в левом меню, прокрутите вниз до раздела **«Your App's Presence in Slack»**, нажмите кнопку «Edit» и задайте `Display Name` (отображаемое имя) и `Default Username` (внутренний ID, только строчные буквы и цифры), затем сохраните. Без этого бота нельзя будет пригласить в канал!

### 2. Включите Socket Mode и получите «App-Level токен»!
1. Откройте **«Socket Mode»** в левом меню.
2. Переведите переключатель **«Enable Socket Mode»** в положение **ВКЛ (зелёный)**.
3. Когда будет запрошено имя токена, введите произвольное название (например, `telopon-socket`) и нажмите «Generate».
4. Появится строка, начинающаяся с **`xapp-`**. Это **[App-Level токен]**. Скопируйте и сохраните её.

### 3. Назначение прав на чтение сообщений (Scopes)
1. Откройте **«OAuth & Permissions»** в левом меню.
2. Прокрутите вниз до раздела **«Scopes»** → **«Bot Token Scopes»**, нажмите «Add an OAuth Scope» и добавьте следующие 5 разрешений:
   * `channels:history` / `groups:history` / `channels:read` / `groups:read` / `users:read`

### 4. Настройка приёма событий в реальном времени (Event Subscriptions)
1. Откройте **«Event Subscriptions»** в левом меню.
2. Переведите переключатель **«Enable Events»** вверху в положение **ВКЛ (зелёный)**.
3. Раскройте раздел **«Subscribe to bot events»** чуть ниже, нажмите «Add Bot User Event», добавьте следующие 2 события и нажмите **«Save Changes»** в правом нижнем углу:
   * `message.channels` / `message.groups`

### 5. Установите в рабочее пространство и получите «Bot токен»!
1. Откройте **«Install App»** в левом меню и нажмите **«Install to Workspace»**. (※ Если вы изменяли права, кнопка будет называться «Reinstall»)
2. На экране разрешений нажмите «Разрешить (Allow)».
3. Появится строка, начинающаяся с **`xoxb-`**. Это **[Bot токен]**. Скопируйте и сохраните её.

### 6. [ОЧЕНЬ ВАЖНО] «Пригласите» бота в канал Slack
Откройте Slack и **в поле ввода канала, который хотите отслеживать (например, #general), отправьте упоминание (@) вашему боту, например `@TeloPon Bot`**. Когда появится вопрос «Добавить в канал?», обязательно подтвердите добавление. Без этого бот не сможет считывать комментарии!

## 🖥️ Шаг 3: Настройка и подключение в интерфейсе TeloPon

Вернитесь в TeloPon и откройте панель настроек (⚙️) у пункта «🏢 Интеграция с комментариями Slack».

![Экран настроек интеграции с Slack](../images/discord_integration.png)

1. **Ввод токенов**
   * В поле «App-Level токен» вставьте строку, начинающуюся с **`xapp-`**, полученную на Шаге 2.
   * В поле «Bot токен» вставьте строку, начинающуюся с **`xoxb-`**, полученную на Шаге 2.
2. **Получение списка каналов**
   * Нажмите кнопку **«🔄 Проверить токены и получить список каналов»**.
   * При успехе будет получен список каналов вашего Slack.
3. **Выберите канал для мониторинга**
   * В выпадающем списке выберите канал для отслеживания. (Открытые каналы отмечены значком 💬, закрытые — значком 🔒)
4. **Подключение!**
   * Наконец, нажмите синюю кнопку **«Подключиться»**. Если статус изменится на зелёный «⚡ Статус: Подключено (Socket Mode работает)», всё готово!

## 💡 Использование и полезные советы

После завершения настройки просто нажмите «Подключиться к трансляции (запуск ИИ)» на главном экране TeloPon! ИИ будет быстро реагировать на сообщения в указанном канале Slack.

* **Автоматическое преобразование имён**: Специфичные для Slack ID пользователей (U1234... и т.д.) автоматически преобразуются в «отображаемое имя отправителя (например, Иван Иванов)» на стороне TeloPon перед передачей ИИ. ИИ будет обращаться к пользователям по имени!
* **Настройка промпта**: Изменив текст инструкции в нижней части экрана настроек на что-то вроде «Это чат от коллег! Отвечай позитивно и дружелюбно!», вы сможете ещё больше раскрыть характер ИИ.
