# 🐦 Плагин X (Twitter) (XTwitterPlugin.py)

📥 **[Скачать XTwitterPlugin.py](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/XTwitterPlugin.py)**

Получение твитов по хэштегу и передача ИИ. ИИ автоматически публикует твиты во время трансляции.

> **Примечание:** Плагин использует платный API X (Twitter). Требуется регистрация в X Developer Portal и оплата API.

---

## 🌟 Основные функции

| Функция | Описание |
|---|---|
| Поиск по хэштегу | Периодическое получение твитов по хэштегу и передача ИИ |
| Публикация твитов | ИИ автоматически публикует твиты по команде стримера |

---

## ⚙️ Требования

- Приложение X (Twitter) Developer Portal (права чтения и записи)
- Библиотека: `pip install tweepy`

---

## 📋 Настройка

1. Создайте приложение в X Developer Portal и получите 4 ключа
2. Введите ключи в настройках плагина → "Тест подключения"
3. Укажите хэштег для поиска и интервал опроса
4. Настройте параметры публикации и хэштеги по умолчанию

---

## ⚠️ Примечания

- **Платный API:** Бесплатный уровень X API отменён. Требуется оплата за использование.
- **Интервал опроса:** Чем короче, тем выше расходы. Рекомендуется 60+ секунд.
