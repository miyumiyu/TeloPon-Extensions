# 🐦 X (Twitter) (XTwitterPlugin.py)

📥 **[Скачать XTwitterPlugin.py](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/XTwitterPlugin.py)**

Получение твитов по хэштегу и передача ИИ. ИИ публикует твиты с текстом, миниатюрой или скриншотом OBS.

> **Примечание:** Требуется платный API X (Twitter). Нужна регистрация в X Developer Portal и оплата Pay Per Use.

---

## 🌟 Основные функции

| Функция | Описание |
|---|---|
| Поиск по хэштегу | Периодическое получение твитов и передача ИИ |
| Текстовая публикация | Автоматические хэштеги и URL трансляции |
| Публикация с миниатюрой | Твит с прикреплённой миниатюрой трансляции |
| Публикация со скриншотом | Твит со скриншотом OBS |

---

## ⚙️ Требования

- Приложение X Developer Portal (добавлено в пакет Pay Per Use)
- Библиотека: `pip install tweepy`

---

## 📋 Настройка

1. Создайте приложение в X Developer Portal → права **Read and Write** → добавьте в **Pay Per Use**
2. Получите 4 ключа + пополните кредит
3. Введите ключи → "Тест подключения"
4. Настройте хэштег, параметры публикации и OBS-захват

---

## 🎤 Голосовые команды

| Команда | Действие |
|---|---|
| "Опубликуй твит" | Текстовый твит |
| "Твит с миниатюрой" | Твит с миниатюрой |
| "Твит с экрана" | Твит со скриншотом OBS |

---

## ⚠️ Примечания

- **Платный API:** Бесплатный уровень отменён. Нужна оплата Pay Per Use.
- **Пакет Pay Per Use:** Без него — ошибка 403.
- **Кредит:** Без баланса — ошибка 402.
