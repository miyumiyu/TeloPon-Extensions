# 📺 Плагин Niconico Live (NiconicoLivePlugin.py)

📥 **[Скачать NiconicoLivePlugin.py](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/NiconicoLivePlugin.py)**

Плагин для интеграции с Niconico Live (ニコニコ生放送). Получает комментарии, подарки, рекламу Niconi, опросы и статистику в реальном времени и передаёт их ИИ.  
Чтение комментариев работает без входа. При авторизации доступны создание опросов и публикация комментариев оператора.

---

## 🌟 Основные функции

### 📖 Базовые функции (без входа)

| Функция | Описание |
|---|---|
| Чтение комментариев | Получение комментариев зрителей в реальном времени |
| Уведомления о подарках | ИИ благодарит за полученные подарки |
| Уведомления о рекламе Niconi | ИИ благодарит за рекламу |
| Уведомления об эмоциях | ИИ реагирует на массовые эмоции |
| Обновление статистики | Автоуведомление при достижении вех по зрителям |
| Результаты опросов | ИИ комментирует результаты опросов |

### ✏️ Функции управления (нужен вход, только своя трансляция)

| Функция | Описание |
|---|---|
| Создание/результаты/закрытие опросов | Управление опросами голосом |
| Комментарии оператора | Публикация (с цветом и закреплением) и удаление |
| Получение числа зрителей | Текущее число зрителей и комментариев |

## 📋 Настройка

### 1. Откройте настройки

Нажмите **"⚙️ Настройки"** рядом с **"Niconico Live Plugin"** на панели расширений TeloPon.

### 3. Подключитесь к Niconico Live

1. Введите URL трансляции (напр., `https://live.nicovideo.jp/watch/lv350259676`)
2. Нажмите **"Подключить"**
3. Если отображается информация о трансляции — подключение успешно

> Для предстоящих трансляций отображается "Ожидание начала..." и автоподключение при старте.

### 4. Вход (опционально)

Для управления опросами и комментариями оператора нужен вход в аккаунт Niconico.

1. Введите email и пароль в разделе "Вход Niconico"
2. Нажмите **"Войти"**
3. При двухфакторной аутентификации появится поле для 6-значного кода
4. После авторизации отображаются иконка профиля и имя

> **Функции управления доступны только для своих трансляций.** Для чужих — только чтение.

## 🎤 Голосовые команды

### Опросы

| Команда | Действие |
|---|---|
| "Создай опрос" | ИИ создаёт опрос с вопросом и вариантами |
| "Покажи результаты" | Отображение результатов опроса |
| "Закрой опрос" | Завершение опроса |

### Комментарии оператора

| Команда | Действие |
|---|---|
| "Напиши комментарий оператора: ○○" | Публикация комментария |
| "Красным: ○○" | Цветной комментарий оператора |
| "Закреплённый: ○○" | Закреплённый комментарий |
| "Удали комментарий оператора" | Удаление комментария |

### Информация

| Команда | Действие |
|---|---|
| "Сколько зрителей?" | Текущее число зрителей и комментариев |

## ⚠️ Примечания

- **Подключение/отключение во время трансляции:** TOOL-тип, подключение/отключение в реальном времени.
- **Управление только своими трансляциями:** Даже при авторизации функции управления недоступны для чужих трансляций.
- **Поддержка 2FA:** Совместимость с двухфакторной аутентификацией Niconico.
