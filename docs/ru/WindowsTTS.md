# 🔊 Озвучка телопов - Windows (WindowsTTS.py)

📥 **[Скачать WindowsTTS.py](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/WindowsTTS.py)**

Плагин для автоматического озвучивания текста баннеров, выводимых ИИ, с использованием стандартного движка синтеза речи Windows (SAPI5).  
Не требует дополнительного внешнего ПО — используются голоса, встроенные в Windows.

---

## 🌟 Основные функции

| Функция | Описание |
|---|---|
| Автоматическое озвучивание баннеров | Автоматическая озвучка текста MAIN, выводимого ИИ |
| Выбор голоса | Выбор из голосов, установленных в Windows (японский/английский и др.) |
| Выбор устройства воспроизведения | Возможность указать устройство вывода (поддержка виртуальных устройств, таких как VB-CABLE) |
| Регулировка скорости | Регулировка скорости озвучивания с помощью ползунка |
| Регулировка высоты тона | Регулировка высоты голоса с помощью ползунка |
| Регулировка громкости | Регулировка громкости озвучивания с помощью ползунка |
| Озвучивание TOPIC | Переключение озвучивания заголовка баннера (TOPIC) |
| Удаление эмодзи | Автоматическое исключение Unicode-эмодзи и текста формата `:emoji:` из озвучивания |

## ⚙️ Настройка

### 1. Откройте экран настроек

На главном экране TeloPon в панели «Расширения» справа нажмите кнопку **«⚙️ Настройки»** у пункта **«Озвучивание Windows»**.

### 2. Настройте параметры

<img src="../images/WindowsTTS.png" width="400">

| Параметр | Описание |
|---|---|
| **Озвучивание ВКЛ** | Установите флажок, чтобы включить озвучивание |
| **Голос** | Выберите голос для озвучивания. Для японского языка часто используется «Microsoft Haruka Desktop - Japanese» |
| **Устройство воспроизведения** | Выберите устройство вывода звука. «Устройство по умолчанию» — стандартный выход, настроенный в Windows |
| **Скорость** | Регулировка скорости озвучивания (100 = медленно ~ 350 = быстро, по умолчанию 200) |
| **Высота тона** | Регулировка высоты голоса (-10 = низкий ~ +10 = высокий, по умолчанию 0) |
| **Громкость** | Регулировка громкости озвучивания (0–100, по умолчанию 100) |
| **Озвучивать TOPIC** | Установите флажок, чтобы озвучивать также заголовок баннера (TOPIC) |

### 3. Тестовое озвучивание

Нажмите кнопку «Тестовое озвучивание», чтобы воспроизвести тестовый звук с текущими настройками.  
Если всё в порядке, нажмите «Закрыть» для сохранения настроек.

---

## 🎧 Как передать звук в OBS

Если вы хотите добавить озвучивание ИИ в трансляцию OBS, используйте виртуальное аудиоустройство.

### Порядок действий

1. Установите виртуальное аудиоустройство, например **VB-CABLE**
2. В плагине WindowsTTS в «Устройстве воспроизведения» выберите виртуальное устройство (например: `CABLE Input`)
3. В OBS добавьте «Захват аудио-входа» и выберите виртуальное устройство (например: `CABLE Output`)

Теперь озвучивание ИИ будет передаваться в OBS и включено в трансляцию.

---

## 📝 Об удалении эмодзи

Эмодзи в баннерах автоматически исключаются из озвучивания.

| Тип | Пример | Поведение |
|---|---|---|
| Unicode-эмодзи | 🎮 🔥 💬 | Удаляются (не озвучиваются) |
| Текстовые эмодзи | `:fire:` `:thumbsup:` | Удаляются (не озвучиваются) |
| Обычный текст | Привет | Озвучивается как есть |

---

## ⚠️ Примечания по использованию

- **Настройки можно менять во время трансляции:** Этот плагин имеет тип TOOL, поэтому скорость, голос, устройство и другие параметры можно изменять в реальном времени во время трансляции.
- **Только для Windows:** SAPI5 — это движок синтеза речи, работающий только на Windows. На macOS/Linux не работает.
- **Добавление голосов:** Дополнительные голосовые пакеты можно установить через «Параметры» → «Время и язык» → «Распознавание речи» в Windows.
