# 💬 Discord (discord_integration.py)

С помощью этого плагина ИИ TeloPon сможет в реальном времени считывать комментарии из указанного канала вашего сервера Discord и реагировать на них!

Настройка состоит из трёх основных шагов: **«① Установка плагина»**, **«② Создание бота на стороне Discord (получение ключа)»** и **«③ Настройка в интерфейсе TeloPon»**. Даже новичок сможет выполнить все шаги за 5 минут!

---

## 📥 Шаг 1: Установка плагина

📥 **[Скачать discord_integration.py](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/discord_integration.py)**

1. Скачайте файл по ссылке выше.
2. Откройте папку `TeloPon-XXX` (основная папка программы) на вашем компьютере.
3. Поместите скачанный файл в папку **`plugins`** внутри неё.
4. Запустите TeloPon (или перезапустите). Если в списке «🔌 Расширения» на главном экране появился пункт **«💬 Интеграция с Discord в реальном времени»**, установка прошла успешно!

---

## 🛠️ Шаг 2: Создание бота Discord и получение «токена»

Чтобы ИИ мог считывать комментарии из Discord, необходимо создать собственного «бота (робота)».

### 1. Откройте Developer Portal
Перейдите в браузере на ПК на [Discord Developer Portal](https://discord.com/developers/applications) и войдите с помощью вашего обычного аккаунта Discord.

### 2. Создание приложения (бота)
1. Нажмите кнопку **«New Application»** в правом верхнем углу.
2. В поле `Name` введите любое имя (например: `TeloPon Bot`), установите флажок согласия с правилами и нажмите **«Create»**.

### 3. [ОЧЕНЬ ВАЖНО] Включите разрешение на чтение сообщений
**※ Если пропустить этот шаг, бот не сможет считывать комментарии! Обязательно выполните настройку.**
1. В левом меню нажмите **«Bot»**.
2. Прокрутите немного вниз и найдите раздел **«Privileged Gateway Intents»**.
3. Переведите переключатель **«Message Content Intent»** в положение **ВКЛ (зелёный)** и, если появится всплывающее окно, нажмите «Save Changes» для сохранения.

### 4. Копирование токена (ключа)
1. Вернитесь к верхней части экрана **«Bot»**.
2. Рядом с пунктом «Token» нажмите кнопку **«Reset Token»** и выберите «Yes, do it!».
3. Появится длинная строка символов (это и есть токен) — нажмите кнопку **«Copy»**, чтобы скопировать её.
*(※ Токен не будет показан повторно после закрытия этой страницы, поэтому вставьте его в блокнот или другое место для хранения)*

## 🖥️ Шаг 3: Настройка и подключение в интерфейсе TeloPon

На главном экране TeloPon откройте «🔌 Расширения» и нажмите кнопку настроек (⚙️) у пункта **«💬 Интеграция с Discord в реальном времени»**, чтобы открыть панель.

![Экран настроек интеграции](../images/slack_integration.png)

### 1. Ввод токена
* В поле «Токен бота» вставьте длинную строку (токен), скопированную на Шаге 2.

### 2. Пригласите бота на свой сервер
* После ввода токена нажмите кнопку **«🌐 Пригласить этого бота на сервер»** чуть ниже.
* Автоматически откроется браузер с экраном приглашения Discord.
* Выберите свой сервер, на который хотите добавить бота, и нажмите «Да» → «Авторизовать».
*(※ Если в Discord появилось сообщение «ХХХ добавил TeloPon Bot», всё прошло успешно!)*

### 3. Получение списка каналов
* Вернитесь в TeloPon и нажмите кнопку **«🔄 Проверить токен и получить список каналов»**.
* При успехе появится сообщение: «✅ Успешно: получено ХХ каналов!»

### 4. Выберите канал для мониторинга
* В выпадающем списке «Канал для мониторинга» ниже отобразятся все каналы вашего сервера (💬 текстовые, 🔊 голосовые и т.д.).
* Выберите канал, из которого ИИ должен считывать сообщения.

### 5. Подключение!
* Наконец, нажмите синюю кнопку **«Запустить (подключиться)»**!
* Если статус изменится на зелёный «⚡ Статус: Подключено», всё готово!

---

## 💡 Использование и полезные советы

После завершения настройки просто нажмите «Подключиться к трансляции (запуск ИИ)» на главном экране TeloPon!
Когда кто-то напишет сообщение в указанном канале Discord, комментарий будет отправлен ИИ практически мгновенно (задержка около 0,1 секунды), и ИИ на него отреагирует.

* **Настройка промпта**: Отредактировав «Текст инструкции для ИИ» в нижней части экрана настроек, вы можете изменить характер реакций ИИ. (Например: «Это комментарии от зрителей из Discord. Отвечай дружелюбно!» и т.п.)
* **Бот становится офлайн?**: Бот в Discord показывает статус «Онлайн (зелёный круг)» только пока нажата кнопка «Подключиться» в TeloPon. При закрытии TeloPon бот автоматически переходит в режим офлайн.
