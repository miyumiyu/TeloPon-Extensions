# 🔊 Озвучка телопов - Gemini TTS (GeminiTTS.py)

📥 **[Скачать GeminiTTS.py](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/GeminiTTS.py)**

Плагин высококачественной озвучки телопов на базе новейшей **Gemini 3.1 Flash TTS Preview**.  
Использует API-ключ Gemini, уже настроенный в TeloPon, — **дополнительная регистрация и аутентификация не требуются**.

---

## 🌟 Основные возможности

| Функция | Описание |
|---|---|
| Автоматическая озвучка телопов | Озвучивает текст MAIN, выводимый ИИ |
| 30 предустановленных голосов | Kore / Puck / Zephyr / Charon / Fenrir и другие |
| Связь с эмоциями (badge) | Стиль чтения меняется в зависимости от эмоционального тега ИИ |
| Указание персонажа (свободный ввод) | Например: «цундэрэ», «весёлая девушка», «спокойный джентльмен» |
| Управление скоростью | Медленно / Обычно / Быстро |
| Выбор устройства вывода | Поддержка виртуальных устройств |
| Громкость | Ползунок 0–100% |
| Задержка | Пауза между отображением телопа и началом озвучки |
| Чтение TOPIC | Переключение: читать ли заголовок TOPIC |
| Выбор модели | Gemini 3.1 Flash TTS Preview / Gemini 2.5 Flash Preview TTS |

---

## ⚠️ Важное предупреждение

> **У Gemini TTS API строгие лимиты. На бесплатном тарифе лимит достигается примерно за 5 вызовов.**
>
> Плагин не предназначен для длительного постоянного использования — используйте его в режиме **ознакомления/демонстрации**.  
> Для серьёзного использования рассмотрите платный тариф Gemini API (pay-as-you-go).

При достижении лимита статус плагина показывает «⚠️ Лимит достигнут (повтор через Xс)», очередь очищается, и плагин ждёт указанное время.

## ⚙️ Настройка

### 1. Предварительные требования
В TeloPon должен быть настроен **API-ключ Gemini**.  
Этот плагин использует его напрямую.

### 2. Открытие настроек
Нажмите **«⚙️ Настройки»** рядом с **«Gemini TTS»** в панели расширений на главном экране TeloPon.

### 3. Параметры

| Пункт | Описание |
|---|---|
| **ON / OFF** | Включение озвучки |
| **Читать TOPIC** | Также читать заголовок TOPIC |
| **Голос** | Выбор одного из 30 предустановленных голосов |
| **Устройство** | Куда выводить звук (виртуальные устройства для OBS) |
| **Персонаж** | Свободный ввод, например: `цундэрэ`, `весёлая девушка`, `спокойный джентльмен` |
| **Скорость** | Медленно / Обычно / Быстро |
| **Громкость** | 0–100% |
| **Задержка** | Секунд от показа телопа до начала чтения (0–10с) |

### 4. Тестовое воспроизведение
Кнопка «▶ Тест» позволяет прослушать выбранные настройки.

## 🎭 Выразительность

Gemini TTS принимает указания на естественном языке.  
То, что вы вводите в поле «Персонаж», передаётся прямо в промпт.

### Пример

```
цундэрэ
→ «в стиле цундэрэ, звучит радостно, в быстром темпе, прочитайте: ТЕКСТ»
```

Эмоциональный тег `[BDG]` от ИИ автоматически объединяется:

- `[BDG]радость` + Персонаж «цундэрэ» + Скорость «быстро»
- → промпт: «в стиле цундэрэ, звучит радостно, в быстром темпе»

Поле «Персонаж» можно оставить пустым — тогда применяются только эмоция и скорость.

## 🎧 Передача звука в OBS

Чтобы вывести TTS-голос в эфир, используйте виртуальное аудиоустройство.

1. Установите **VB-CABLE** (или аналогичное виртуальное устройство)
2. В «Устройстве» плагина выберите виртуальный вход (например, `CABLE Input`)
3. В OBS добавьте источник «Захват входного аудио» и выберите виртуальный выход (например, `CABLE Output`)

## 💡 Как выбрать голос

Доступно 30 предустановленных голосов. Каждый работает на японском, английском и других языках с естественным произношением.

Популярные голоса:
- **Kore** — спокойный женский голос (по умолчанию)
- **Puck** — яркий, живой
- **Zephyr** — мягкий, ровный
- **Charon** — глубокий мужской голос
- **Fenrir** — мощный мужской голос

Используйте кнопку «Тест» для сравнения и выбора.

## ⚠️ Примечания по использованию

- **Редактируется в эфире:** Это плагин типа TOOL — голос, указание персонажа, скорость и т.д. можно менять в реальном времени во время стрима.
- **Следите за лимитами:** На бесплатном тарифе быстрое повторное использование приводит к ограничениям. См. предупреждение выше.
- **Нет стриминга:** Gemini TTS API не поддерживает потоковое воспроизведение, поэтому задержка между выводом телопа и озвучкой составляет 1–3 секунды.
