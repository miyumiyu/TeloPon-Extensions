# 📊 PowerPoint 조작 플러그인 (PowerPointPlugin.py)

📥 **[PowerPointPlugin.py 다운로드](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/PowerPointPlugin.py)**

방송 중 음성 명령으로 PowerPoint 슬라이드쇼를 조작할 수 있는 플러그인입니다.  
슬라이드 넘기기/되돌리기, 페이지 점프, 블랙아웃 등을 지원하며, 슬라이드가 바뀌면 제목과 노트를 AI에 자동 주입합니다.

---

## 🌟 주요 기능

| 기능 | 설명 |
|---|---|
| 다음/이전 슬라이드 | "다음 슬라이드", "이전으로"로 조작 |
| 페이지 점프 | "3페이지로 이동해"로 특정 슬라이드로 이동 |
| 처음/마지막으로 이동 | "처음으로 돌아가", "마지막 슬라이드"로 이동 |
| 프레젠테이션 시작/종료 | "프레젠테이션 시작해", "프레젠테이션 끝내"로 제어 |
| 화면 블랙아웃 | "화면 꺼줘"로 암전, "화면 복구해"로 복귀 |
| 슬라이드 노트 주입 | 슬라이드 변경 시 제목과 노트를 AI에 자동 전송 |

---

## ⚙️ 필요 환경

- Windows PC
- Microsoft PowerPoint 설치 필요
- pywin32 (TeloPon exe 버전에 포함)

---

## 📋 설정 방법

### 1. 설정 화면 열기

TeloPon 메인 화면 오른쪽 "확장 기능" 패널에서 **"PowerPoint 조작"**의 **"⚙️ 설정"** 버튼을 클릭합니다.

### 2. PowerPoint에 연결

1. 조작할 PowerPoint 파일을 미리 열어둡니다
2. 설정 화면에서 **"PowerPoint에 연결"** 버튼을 클릭
3. "연결됨: 파일명.pptx"가 표시되면 성공입니다

### 3. 옵션 설정

| 항목 | 설명 |
|---|---|
| **PowerPoint 연동 ON** | 체크하면 플러그인이 활성화됩니다 |
| **슬라이드 노트를 AI에 주입** | ON으로 하면 슬라이드 변경 시 발표자 노트를 AI에 전송합니다 |
| **슬라이드 제목을 AI에 주입** | ON으로 하면 슬라이드 제목을 AI에 전송합니다 |

### 4. 테스트 버튼

"다음 (테스트)", "이전 (테스트)" 버튼으로 슬라이드쇼 중 동작을 확인할 수 있습니다.

---

## 🎤 음성 명령 목록

방송 중 다음과 같이 말하면 AI가 슬라이드를 조작합니다.

| 음성 예시 | 동작 |
|---|---|
| "다음 슬라이드" / "다음으로" | 다음 슬라이드로 |
| "이전 슬라이드" / "뒤로 가줘" | 이전 슬라이드로 |
| "3페이지로 이동해" / "5번으로 가줘" | 지정 페이지로 점프 |
| "처음으로 돌아가" | 첫 슬라이드로 |
| "마지막 슬라이드" | 마지막 슬라이드로 |
| "프레젠테이션 시작해" | 슬라이드쇼 시작 |
| "프레젠테이션 끝내" | 슬라이드쇼 종료 |
| "화면 꺼줘" / "블랙아웃" | 화면 암전 |
| "화면 복구해" | 암전에서 복귀 |

---

## 📝 슬라이드 노트의 AI 주입

슬라이드가 전환될 때마다 다음 정보가 AI에 자동 전송됩니다.

```
[프레젠테이션 노트]
슬라이드 3/15: 슬라이드 제목
발표자 노트 텍스트가 여기에 표시됩니다...
```

이를 통해 AI가 현재 슬라이드의 내용을 파악하고, 프레젠테이션 흐름에 맞는 리액션을 할 수 있습니다.  
발표자 노트에 "이 포인트를 강조", "여기서 유머를" 등의 메모를 작성해두면 AI가 참고하여 반응합니다.

## ⚠️ 이용 시 주의사항

- **라이브 중 연결/해제 가능:** TOOL 타입 플러그인이므로 방송 중에도 PowerPoint 연결/해제를 실시간으로 할 수 있습니다.
- **Windows 전용:** COM을 통해 PowerPoint를 조작하므로 Windows에서만 작동합니다.
- **PowerPoint를 먼저 실행:** 연결 전에 PowerPoint에서 파일을 열어둬야 합니다.
- **슬라이드쇼 모드:** 슬라이드 조작 명령은 슬라이드쇼 모드 중에만 유효합니다.
- **시청자 댓글에서의 조작:** 보안상 시청자 댓글로는 슬라이드 조작이 불가합니다 (스트리머 음성만 가능).
