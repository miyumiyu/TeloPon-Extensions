# 🔊 텔롭 읽기 - Gemini TTS (GeminiTTS.py)

📥 **[GeminiTTS.py 다운로드](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/GeminiTTS.py)**

최신 **Gemini 3.1 Flash TTS Preview**를 사용하여 AI가 출력한 텔롭을 고품질 음성으로 읽어주는 플러그인입니다.  
TeloPon 본체에 설정된 Gemini API 키를 그대로 사용하므로 **별도의 계정 등록이나 인증이 불필요**합니다.

---

## 🌟 주요 기능

| 기능 | 설명 |
|---|---|
| 텔롭 자동 읽기 | AI가 출력한 MAIN 텍스트를 자동으로 음성화 |
| 30가지 프리셋 보이스 | Kore / Puck / Zephyr / Charon / Fenrir 등 다양한 캐릭터 |
| 감정(badge) 연동 | AI가 부여한 감정 태그에 따라 읽는 방식이 변화 |
| 캐릭터 지시(자유 입력) | "츤데레풍" "밝은 여자아이" "차분한 신사" 등 연출 지시 |
| 속도 조정 | 느리게 / 보통 / 빠르게 |
| 출력 장치 선택 | 출력 장치 지정 가능(가상 장치 대응) |
| 음량 조정 | 0〜100% 슬라이더 |
| 딜레이 설정 | 텔롭 표시부터 읽기 시작까지 대기 시간 |
| TOPIC 읽기 | 제목(TOPIC)도 읽을지 여부 전환 |
| 모델 선택 | Gemini 3.1 Flash TTS Preview / Gemini 2.5 Flash Preview TTS |

---

## ⚠️ 중요한 주의 사항

> **Gemini TTS API는 레이트 리미트가 엄격하며, 무료 API로는 5회 정도에서 리미트에 도달합니다.**
>
> 장시간 방송 상시 사용에는 적합하지 않으므로, **체험·데모용**으로 사용해 주세요.  
> 본격 사용 시에는 Gemini API의 유료 플랜(종량제)을 검토해 주세요.

레이트 리미트에 도달하면 플러그인 상태에 "⚠️ 리미트 도달 (○초 후 재시도)"가 표시되며, 자동으로 큐를 비우고 지정 시간 대기합니다.

## ⚙️ 설정 방법

### 1. 전제 조건
TeloPon 본체에서 **Gemini API 키**가 설정되어 있어야 합니다.  
이 플러그인은 본체의 설정을 그대로 사용합니다.

### 2. 설정 화면 열기
TeloPon 메인 화면 우측의 "확장 기능" 패널에서 **"Gemini TTS"**의 **"⚙️ 설정"** 버튼을 클릭합니다.

### 3. 각 항목 설정

| 항목 | 설명 |
|---|---|
| **ON / OFF** | 체크하면 읽기가 활성화 |
| **TOPIC도 읽기** | 텔롭 제목(TOPIC)도 읽을 경우 ON |
| **음성** | 30가지 프리셋 보이스에서 선택 |
| **출력 장치** | 음성 출력 대상(VB-CABLE 등 가상 장치 선택 가능) |
| **캐릭터 지시** | 자유 입력. 예: `츤데레풍`, `밝은 여자아이`, `차분한 신사` 등 |
| **속도** | 느리게 / 보통 / 빠르게 |
| **음량** | 0〜100% |
| **딜레이** | 텔롭 표시부터 읽기 시작까지 초수(0〜10초) |

### 4. 테스트 재생
"▶ 테스트" 버튼으로 설정 내용의 테스트 음성을 확인할 수 있습니다.

## 🎭 표현력에 대하여

Gemini TTS는 자연어 연출 지시에 대응합니다.  
캐릭터 지시에 입력한 내용이 그대로 프롬프트에 반영됩니다.

### 캐릭터 지시 예시

```
츤데레풍
→ 「츤데레풍, 기쁜 느낌으로, 빠르게 읽어주세요: 텍스트」
```

AI가 부여한 `[BDG]` 감정 태그도 자동으로 조합됩니다:

- `[BDG]기쁨` + 캐릭터 "츤데레풍" + 속도 "빠르게"
- → "츤데레풍, 기쁜 느낌으로, 빠르게" 프롬프트

캐릭터 지시는 공란이어도 동작합니다(그 경우 감정·속도만 반영).

## 🎧 OBS에 음성 통합하기

방송에 AI 읽기 음성을 얹고 싶은 경우, 가상 오디오 장치를 사용합니다.

1. **VB-CABLE** 등 가상 오디오 장치 설치
2. 플러그인의 "출력 장치"에서 가상 장치(예: `CABLE Input`) 선택
3. OBS에서 "오디오 입력 캡처" 추가, 가상 장치(예: `CABLE Output`) 선택

## 💡 보이스 선택 방법

30가지 프리셋에서 선택 가능합니다. 한국어·일본어·영어 모두 자연스러운 발음으로 읽습니다.

대표적인 보이스:
- **Kore** — 차분한 여성 목소리(기본값)
- **Puck** — 밝고 활발한 목소리
- **Zephyr** — 부드럽고 온화한 목소리
- **Charon** — 깊이 있는 남성 목소리
- **Fenrir** — 강인한 남성 목소리

원하는 보이스가 찾기 어려울 경우, 테스트 재생 버튼으로 순서대로 비교해 보세요.

## ⚠️ 사용 시 주의

- **라이브 중 설정 변경 가능:** TOOL 타입이므로 방송 중에 음성·캐릭터 지시·속도 등을 실시간 변경할 수 있습니다.
- **레이트 리미트 주의:** 무료 플랜에서는 짧은 시간에 여러 번 발동하면 제한됩니다. 위 주의 사항 참조.
- **스트리밍 비지원:** Gemini TTS API는 스트리밍 재생에 대응하지 않으므로, 텔롭 출력부터 음성 재생까지 1〜3초 지연이 있습니다.
