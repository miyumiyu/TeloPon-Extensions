# 📺 YouTube Live+ (YoutubeLiveOAuth.py)

📥 **[YoutubeLiveOAuth.py 다운로드](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/YoutubeLiveOAuth.py)**

YouTube Data API v3 + OAuth2를 활용한 고기능 YouTube Live 연동 플러그인입니다.  
댓글 읽기/쓰기, 설문조사, 제목 변경, 시청자 수 조회 등을 방송자의 음성으로 조작할 수 있습니다.

> 기존 「YouTube 연동 도구 (YoutubeLivePlugin.py)」의 상위 호환입니다.  
> 기존 버전은 댓글 읽기만 가능했지만, 본 플러그인은 OAuth2 인증을 통해 쓰기 및 각종 조작이 가능합니다.

---

## 🌟 주요 기능

| 기능 | 설명 |
|---|---|
| 채팅 댓글 읽기 | 시청자의 댓글을 실시간으로 AI에 주입 |
| 채팅 댓글 쓰기 | AI가 YouTube 채팅에 게시 |
| 설문조사(Poll) 생성/마감/집계 | YouTube 공식 설문조사 기능을 음성으로 조작 |
| 방송 제목 변경 | 음성 지시로 제목을 변경 |
| 시청자 수 조회 | 현재 동시 시청자 수를 조회하여 AI가 알려줌 |
| 방송 선택(목록/URL) | 예약된 방송 목록에서 선택하거나 URL로 직접 지정 |
| 권한 자동 판별 | 소유자/관리자 여부를 자동 판별하여 사용 가능한 기능을 표시 |

연결하는 방송에 대해 어떤 권한을 보유하고 있는지에 따라 사용 가능한 기능이 달라집니다.

| 기능 | 소유자/관리자 | 시청자/모더레이터 |
|---|---|---|
| 댓글 읽기 | ✅ | ✅ |
| 댓글 쓰기 | ✅ | ✅ |
| 시청자 수 조회 | ✅ | ✅ |
| 설문조사 | ✅ | ❌ |
| 제목 변경 | ✅ | ❌ |

> 권한은 연결 시 자동으로 판별되며, 방송 정보 미리보기에 표시됩니다.  
> 사용할 수 없는 기능은 자동으로 체크가 해제되고 비활성화됩니다.

## ⚙️ 연동까지의 절차

### Step 1: Google Cloud Console 설정 (최초 1회만)

최초에는 Google Cloud Console에서 프로젝트 생성, API 활성화, 클라이언트 ID 취득이 필요합니다.  
이미지가 포함된 상세한 설정 가이드를 준비했으므로, 해당 가이드를 따라 진행해 주세요.

👉 **[Google Cloud Console 설정 가이드 (이미지 포함)](YoutubeLiveOAuth_GCP_Setup.md)**

가이드 절차:
1. Google Cloud Console에 로그인
2. 새 프로젝트 생성
3. OAuth 동의 화면 설정
4. YouTube Data API v3 활성화
5. 범위(Scope) 설정
6. 테스트 사용자 추가
7. OAuth 클라이언트 ID 생성 → **Client ID**와 **Client Secret** 취득
8. TeloPon에 입력하여 인증

여기까지 완료하면 TeloPon의 설정 화면에 채널명과 썸네일이 표시됩니다.  
2회째부터는 인증이 완료된 상태이므로 이 절차는 건너뛸 수 있습니다.

---

### Step 2: 방송을 선택하여 연결하기

인증 후, 방송을 선택하는 방법은 2가지가 있습니다.

#### 방법A: 방송 목록에서 선택

1. 「방송 목록 가져오기」 버튼을 누릅니다 (예약된 방송이 목록으로 표시됩니다)
2. 목록에서 방송을 선택 → 제목과 썸네일이 미리보기로 표시됩니다
3. 「이 방송에 연결」 버튼을 누릅니다

> ⚠️ **방송 목록에는 인증을 수행한 계정 자신의 방송만 표시됩니다.**  
> 다른 채널의 관리자 권한을 보유하고 있더라도, 해당 채널의 방송은 목록에 표시되지 않습니다.  
> 관리자 권한으로 다른 채널에 연결하려면, 아래의 **방법B (URL로 직접 지정)**를 사용해 주세요.

#### 방법B: URL로 직접 지정

1. YouTube Live의 URL을 입력란에 붙여넣기
2. 「확인」 버튼을 누릅니다 → 제목과 썸네일이 미리보기로 표시됩니다
3. 「이 방송에 연결」 버튼을 누릅니다

> 어떤 방법이든 연결 시 권한이 자동으로 판별됩니다.  
> `권한: 소유자 (전체 기능 사용 가능)` 또는 `권한: 시청자 (댓글/시청자 수만 가능)` 로 표시됩니다.

---

### Step 3: 라이브 방송 시작하기

TeloPon의 메인 화면에서 **「🔴 라이브 연결 시작」**을 누릅니다.

> **중요:** 라이브 방송을 시작하기 **전에** 「이 방송에 연결」을 눌러 두세요.  
> 먼저 연결해 두면 AI에게 방송 정보와 명령어 설명이 확실하게 전달됩니다.  
> 라이브 시작 후에 연결하는 것도 가능하지만, 명령어 인식 정확도가 다소 낮아질 수 있습니다.

## 🎙️ 발화 예시 (이렇게 말하면 AI가 반응합니다)

### 댓글 쓰기

방송자가 AI에게 YouTube 채팅에 글을 쓰도록 지시할 수 있습니다.

| 방송자의 발화 | AI의 동작 |
|---|---|
| 「채팅에 "안녕하세요"라고 써줘」 | YouTube 채팅에 「안녕하세요」라고 게시합니다 |
| 「"오늘도 잘 부탁해"라고 댓글 달아줘」 | YouTube 채팅에 「오늘도 잘 부탁해」라고 게시합니다 |

### 설문조사 ※소유자/관리자 전용

YouTube의 공식 설문조사 기능(Poll)을 사용하여 시청자에게 투표를 받을 수 있습니다.  
선택지는 2~4개까지 설정할 수 있습니다.

| 방송자의 발화 | AI의 동작 |
|---|---|
| 「좋아하는 색을 설문조사해줘」 | 「좋아하는 색은? 빨강 / 파랑 / 초록」과 같은 설문조사가 채팅에 표시됩니다 |
| 「다음에 뭐 할지 설문조사 해줘」 | AI가 적절한 선택지를 생각하여 설문조사를 생성합니다 |
| 「집계해줘」「설문조사 마감해줘」 | 설문조사를 마감하고 AI가 결과를 시청자에게 발표합니다 |

> 설문조사를 마감하면 AI에게 집계 결과가 피드백됩니다.  
> AI가 「빨강이 15표로 1위입니다!」와 같이 자막으로 결과를 발표합니다.

### 제목 변경 ※소유자/관리자 전용

| 방송자의 발화 | AI의 동작 |
|---|---|
| 「제목을 "잡담 방송"으로 바꿔줘」 | 방송 제목이 「잡담 방송」으로 변경됩니다 |
| 「제목 바꿔줘, 마크 건축 회로 해줘」 | 방송 제목이 「마크 건축 회」로 변경됩니다 |

### 시청자 수 조회

| 방송자의 발화 | AI의 동작 |
|---|---|
| 「지금 몇 명이 보고 있어?」 | AI가 현재 동시 시청자 수를 조회하여 자막으로 알려줍니다 |
| 「시청자 수 알려줘」 | 위와 동일 |

## 🛡️ 명령어 시청자 허용 설정

각 기능에는 「시청자 허용」 체크박스가 있습니다.  
**시청자의 댓글로도 AI에게 명령을 실행시킬지 여부**를 제어할 수 있습니다.

| 설정 | 동작 |
|---|---|
| ☑ 시청자 허용 ON | 시청자가 「설문조사해줘」 등의 댓글을 작성하면 AI가 실행할 수 있습니다 |
| ☐ 시청자 허용 OFF | 방송자의 음성 지시로만 실행됩니다. 시청자 댓글로는 실행하지 않습니다 |

> 제목 변경 등 영향이 큰 작업을 시청자에게 허용할지는 신중하게 판단해 주세요.

## 🔧 기능 설정

설정 화면의 오른쪽에 기능 ON/OFF 체크박스가 있습니다.

| 기능 | 기본값 | 설명 |
|---|---|---|
| 댓글 가져오기 | ☑ ON | 시청자의 댓글을 AI에게 전송 |
| 댓글 쓰기 | ☑ ON | AI가 채팅에 게시 가능 |
| 설문조사 | ☑ ON | 설문조사 생성/마감 |
| 제목 변경 | ☑ ON | 방송 제목 변경 |

- 라이브 방송 중에는 기능 설정이 비활성화되어 변경할 수 없습니다 (주입된 프롬프트는 취소할 수 없으므로)
- 권한이 없는 기능은 자동으로 체크가 해제되고 비활성화됩니다

## 📊 2열 레이아웃 설정 화면

설정 화면은 좌우 2열 레이아웃으로 구성되어 있습니다.

### 왼쪽 열

| 섹션 | 내용 |
|---|---|
| OAuth2 인증 | Client ID / Secret 입력, 인증 버튼, 사용자 정보 표시 |
| 방송 선택 | 방송 목록 가져오기, URL 입력, 확인/연결 버튼 |
| 방송 정보 미리보기 | 제목, 설명, 썸네일, 권한 표시 |

### 오른쪽 열

| 섹션 | 내용 |
|---|---|
| 기능 설정 | 각 기능의 ON/OFF + 시청자 허용 체크 |
| 닫기 버튼 | 설정을 저장하고 화면 닫기 |

## ⚠️ 이용 시 주의사항

### 연결 타이밍

라이브 방송을 시작하기 **전에** 「이 방송에 연결」을 눌러 두는 것을 권장합니다.  
라이브 시작 후에 연결한 경우에도 명령어는 사용할 수 있지만, AI에게 전달되는 방식이 약간 다릅니다.

### OAuth 동의 화면의 테스트 모드

Google Cloud Console에서 생성한 OAuth 앱은 「테스트 모드」인 채로 사용할 수 있습니다.  
테스트 사용자에 자신의 이메일 주소를 추가하면 공개 심사는 불필요합니다.  
테스트 모드에서는 최대 100명의 테스트 사용자를 등록할 수 있습니다.

### 관리자 권한으로 연결

방송 채널의 소유자가 아니더라도, YouTube Studio에서 「관리자」로 초대받았다면 전체 기능을 사용할 수 있습니다.  
단, 인증 시 관리 대상의 Brand Account를 선택하여 로그인해야 합니다.

### API 할당량

YouTube Data API v3에는 1일 10,000 유닛의 할당량 제한이 있습니다.  
일반적인 방송에서는 충분한 양이지만, 대량의 API 호출이 발생하는 경우 주의해 주세요.

### 기존 YouTube 연동 도구와의 병용

기존 버전(YoutubeLivePlugin.py)과 본 플러그인(YoutubeLiveOAuth.py)을 동시에 활성화하면,  
댓글이 AI에 이중으로 전송됩니다. 둘 중 하나만 활성화해 주세요.
