# 💬 Discord 실시간 연동 (discord_integration.py)

이 플러그인을 사용하면, Discord 서버의 지정한 채널의 댓글을 TeloPon의 AI가 실시간으로 읽고 반응해 줍니다!

설정은 크게 **「① 플러그인 설치」**, **「② Discord 측에서 Bot 생성 (키 취득)」**, **「③ TeloPon 화면에서 설정」**의 3단계입니다. 초보자도 순서대로 진행하면 5분이면 완료됩니다!

---

## 📥 Step 1: 플러그인 설치

📥 **[discord_integration.py 다운로드](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/discord_integration.py)**

1. 위 링크에서 파일을 다운로드합니다.
2. 사용 중인 컴퓨터에 있는 `TeloPon-XXX` 폴더 (본체 폴더)를 엽니다.
3. 그 안에 있는 **`plugins`** 폴더에 다운로드한 파일을 그대로 넣습니다.
4. TeloPon을 실행 (또는 재시작)합니다. 메인 화면의 「🔌 확장 기능」 목록에 **「💬 Discord 실시간 연동」**이 추가되어 있으면 설치 성공입니다!

---

## 🛠️ Step 2: Discord Bot을 생성하고 「토큰」을 취득하기

AI가 Discord의 댓글을 읽으려면, 본인 전용의 「Bot(로봇)」을 생성해야 합니다.

### 1. Developer Portal에 접속
PC 브라우저에서 [Discord Developer Portal](https://discord.com/developers/applications)에 접속하여, 평소 사용하는 Discord 계정으로 로그인합니다.

### 2. 애플리케이션(Bot) 생성
1. 화면 오른쪽 상단의 **「New Application」** 버튼을 클릭합니다.
2. `Name`에 원하는 이름 (예: `TeloPon 읽기 Bot`)을 입력하고, 약관 체크박스에 체크한 후 **「Create」**를 누릅니다.

### 3. [매우 중요] 메시지를 읽는 권한 ON으로 설정
**※이것을 잊으면 댓글을 읽을 수 없습니다! 반드시 설정해 주세요.**
1. 왼쪽 메뉴에서 **「Bot」**을 클릭합니다.
2. 화면을 약간 아래로 스크롤하여 **「Privileged Gateway Intents」** 항목을 찾습니다.
3. 그 안에 있는 **「Message Content Intent」** 스위치를 **ON(녹색)**으로 설정하고, 팝업이 나오면 「Save Changes」를 눌러 저장합니다.

### 4. 토큰(키) 복사
1. 같은 **「Bot」** 화면의 상단으로 돌아갑니다.
2. 「Token」 항목의 오른쪽에 있는 **「Reset Token」** 버튼을 누르고, 「Yes, do it!」을 선택합니다.
3. 긴 문자열 (이것이 토큰입니다)이 표시되면 **「Copy」** 버튼을 눌러 복사합니다.
*(※이 화면을 닫으면 다시 표시되지 않으므로, 메모장 등에 붙여넣어 두세요)*

---

## 🖥️ Step 3: TeloPon UI에서 설정 및 연결

TeloPon의 메인 화면에서 「🔌 확장 기능」을 열고, **「💬 Discord 실시간 연동」**의 설정(⚙️) 버튼을 눌러 패널을 엽니다.

![Discord 연동 설정 화면](../images/slack_integration.png)

### 1. 토큰 입력
* 「Bot 토큰」 입력란에 Step 2에서 복사한 긴 문자열(토큰)을 붙여넣습니다.

### 2. Bot을 자신의 서버에 초대하기
* 토큰을 입력한 상태에서, 바로 아래에 있는 **「🌐 이 Bot을 서버에 초대하기」** 버튼을 클릭합니다.
* 자동으로 브라우저가 열리고 Discord의 초대 화면이 표시됩니다.
* Bot을 추가할 자신의 서버를 선택하고, 「예」→「인증」으로 진행합니다.
*(※Discord 화면에 「○○가 TeloPon 읽기 Bot을 추가했습니다」라고 나오면 성공입니다!)*

### 3. 채널 목록 가져오기
* TeloPon 화면으로 돌아가서, **「🔄 토큰 확인 & 채널 목록 가져오기」** 버튼을 누릅니다.
* 성공하면 「✅ 성공: ○개의 채널을 가져왔습니다!」라는 메시지가 나옵니다.

### 4. 모니터링할 채널 선택
* 아래의 「모니터링할 채널」 드롭다운(▼)을 열면, 서버의 채널 목록 (💬텍스트, 🔊음성 등)이 전부 표시됩니다.
* 읽어줄 채널을 선택합니다.

### 5. 연결!
* 마지막으로 파란색 **「시작하기(연결)」** 버튼을 누릅니다!
* 상태가 「⚡ 상태: 연결 중」의 녹색으로 바뀌면 준비 완료입니다!

---

## 💡 사용법 및 팁

설정이 완료되면, 평소대로 TeloPon의 메인 화면에서 「라이브 방송에 연결(AI 시작)」을 누르기만 하면 됩니다!
지정한 Discord 채널에서 누군가 발언하면, 약 0.1초의 지연으로 즉시 AI에게 댓글이 전달되어 AI가 반응해 줍니다.

* **프롬프트 커스터마이징**: 설정 화면 하단에 있는 「AI에게 전달할 지시 텍스트」를 수정하면, AI의 반응 방식을 변경할 수 있습니다. (예: 「Discord 리스너의 댓글입니다. 친근하게 답해줘!」 등)
* **Bot이 오프라인이 됨?**: TeloPon의 「연결」 버튼을 누르고 있는 동안에만 Discord에서 Bot이 「온라인(녹색 원)」으로 표시됩니다. TeloPon을 종료하면 자동으로 오프라인으로 돌아갑니다.
