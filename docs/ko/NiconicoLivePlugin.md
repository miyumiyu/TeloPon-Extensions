# 📺 니코니코 생방송 연동 플러그인 (NiconicoLivePlugin.py)

📥 **[NiconicoLivePlugin.py 다운로드](https://raw.githubusercontent.com/miyumiyu/TeloPon-Extensions/main/plugins/NiconicoLivePlugin.py)**

니코니코 생방송의 댓글·기프트·니코니 광고·앙케이트·통계 정보를 실시간으로 가져와 AI에 주입하는 플러그인입니다.  
로그인 없이도 댓글 읽기가 가능하며, 로그인하면 앙케이트 생성이나 운영 댓글 게시 등의 조작 기능도 사용할 수 있습니다.

---

## 🌟 주요 기능

### 📖 기본 기능 (로그인 불필요)

| 기능 | 설명 |
|---|---|
| 댓글 읽기 | 시청자 댓글을 실시간으로 가져와 AI에 주입 |
| 기프트 알림 | 기프트가 도착하면 AI가 감사 인사 |
| 니코니 광고 알림 | 광고가 들어오면 AI가 감사 표현 |
| 이모션 알림 | 이모션이 대량으로 보내지면 AI가 반응 |
| 통계 정보 업데이트 | 방문자 수가 마일스톤을 넘으면 자동 알림 |
| 앙케이트 결과 알림 | 앙케이트 결과를 AI가 코멘트 |

### ✏️ 조작 기능 (로그인 필요, 자기 방송만)

| 기능 | 설명 |
|---|---|
| 앙케이트 생성/결과/종료 | 음성 지시로 앙케이트를 생성·집계·종료 |
| 운영 댓글 게시/삭제 | 운영 댓글 게시 (색상·고정 지원) 및 삭제 |
| 시청자 수 가져오기 | 현재 동시 시청자 수와 댓글 수 조회 |

## ⚙️ 필요 환경

- Windows / macOS / Linux
- 다음 라이브러리가 필요합니다 (TeloPon exe 버전에 미포함):

```
pip install aiohttp websockets protobuf
```

> **주의:** `_niconico` 폴더도 `plugins/` 폴더에 함께 복사해야 합니다.

## 📋 설정 방법

### 1. 파일 배치

다음 2가지를 TeloPon의 `plugins/` 폴더에 복사합니다:
- `NiconicoLivePlugin.py`
- `_niconico/` 폴더 (전체)

### 2. 설정 화면 열기

TeloPon 메인 화면 오른쪽 "확장 기능" 패널에서 **"니코니코 생방송 연동"**의 **"⚙️ 설정"** 버튼을 클릭합니다.

### 3. 니코니코 생방송에 연결

1. 니코니코 생방송 URL을 입력합니다 (예: `https://live.nicovideo.jp/watch/lv350259676`)
2. **"연결"** 버튼을 클릭
3. 방송 정보 (제목·설명문·썸네일)가 표시되면 성공입니다

> 아직 시작되지 않은 방송은 "방송 시작 대기 중"으로 표시되며, 방송이 시작되면 자동 연결됩니다.

### 4. 로그인 (선택 사항)

앙케이트 생성이나 운영 댓글 게시 등의 조작 기능을 사용하려면 니코니코 계정 로그인이 필요합니다.

1. 설정 화면 상단 "니코니코 로그인" 섹션에 이메일과 비밀번호를 입력
2. **"로그인"** 버튼을 클릭
3. 2단계 인증이 활성화된 경우 확인 코드 (6자리) 입력 화면이 표시됩니다
4. 인증이 완료되면 프로필 아이콘과 이름이 표시됩니다

> **조작 기능은 자기 방송에서만 사용 가능합니다.** 다른 사람의 방송에서는 읽기 전용이 됩니다.

### 5. 기능 ON/OFF 설정

오른쪽 컬럼에서 각 기능을 개별적으로 ON/OFF할 수 있습니다.

## 🎤 음성 명령 목록

### 앙케이트

| 음성 예시 | 동작 |
|---|---|
| "앙케이트 해줘" / "투표 해줘" | AI가 질문과 선택지를 만들어 앙케이트 생성 |
| "집계해줘" / "결과 알려줘" | 앙케이트 결과 표시 |
| "앙케이트 마감해" | 앙케이트 종료 |

### 운영 댓글

| 음성 예시 | 동작 |
|---|---|
| "운영 댓글로 ○○ 표시해" | 운영 댓글 게시 |
| "빨간색으로 ○○" | 색상 운영 댓글 |
| "고정으로 ○○" | 고정 운영 댓글 |
| "운영 댓글 삭제해" | 운영 댓글 삭제 |

### 정보

| 음성 예시 | 동작 |
|---|---|
| "지금 몇 명?" / "시청자 수는?" | 동시 시청자 수와 댓글 수 조회 |

## ⚠️ 이용 시 주의사항

- **라이브 중 연결/해제 가능:** TOOL 타입 플러그인이므로 방송 중에도 실시간 연결/해제 가능합니다.
- **조작 기능은 자기 방송만:** 로그인해도 다른 사람의 방송에서는 조작 기능이 비활성화됩니다.
- **2단계 인증 지원:** 니코니코 계정의 2FA에 대응합니다.
- **추가 라이브러리 필요:** `aiohttp`, `websockets`, `protobuf`를 별도로 설치해야 합니다.
- **`_niconico` 폴더 필수:** 플러그인 파일뿐만 아니라 `_niconico/` 폴더도 함께 복사하세요.
